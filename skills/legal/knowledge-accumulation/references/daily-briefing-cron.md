# 每日法律速报 Cron Job 配置

## 参数

| 参数 | 值 |
|------|-----|
| 名称 | 每日法律速报 |
| Cron | `0 8 * * *`（每天8:00） |
| 技能 | `legal-research` |
| 投递 | `origin`（飞书当前聊天） |
| 重复 | `forever` |

## Prompt模板

```
你是余正洪律师的AI法律助手。请执行以下任务：

1. 搜索今日法律热点新闻（最高人民法院、司法部、全国人大等权威来源）
2. 搜索与余律师执业领域相关的最新案例（民商事诉讼、刑事诉讼、知识产权、公司法）
3. 搜索江苏省内法院的最新动态

输出格式：
📋 每日法律速报 — [日期]

🔥 今日热点（3-5条）
⚖️ 最新案例（2-3个）
📢 法规动态（如有）
💡 实务提示

将结果保存到：法律知识库/09-每日法律速报/[YYYY-MM-DD].md
```

## 已知限制

> [!pitfall] MCP工具不返回实时新闻
> 北大法宝和元典智库返回的是历史法律数据库内容，不是今日热点新闻。
> Prompt中需要Agent同时使用web_search获取实时新闻，MCP工具用于案例和法规检索。

## 管理指令

| 用户说 | 动作 |
|--------|------|
| "暂停每日速报" | `cronjob action=pause` |
| "恢复每日速报" | `cronjob action=resume` |
| "改到每天X点" | `cronjob action=update schedule="0 X * * *"` |
| "速报加上XXX" | `cronjob action=update prompt=...` |
| "现在跑一次" | `cronjob action=run` |
