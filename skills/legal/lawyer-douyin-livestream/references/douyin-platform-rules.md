# 抖音直播平台规则与律师合规要点

## 一、抖音直播内容审核机制

### 1.1 审核层级
- **AI 初审**：实时扫描直播间语音转文字+弹幕关键词
- **人工复审**：AI 标记后由人工判断是否违规
- **举报处理**：用户举报触发专项审核

### 1.2 触发审核的关键词
抖音采用模糊匹配+语义理解双重机制，以下内容会被标记：
- 承诺性语言（保证、一定、包、绝对）
- 引流到站外（加微信、加QQ、私聊链接）
- 涉及金钱交易（付费咨询、收费标准）
- 敏感政治话题
- 低俗/暴力内容

---

## 二、律师直播特殊规则

### 2.1 抖音对法律类内容的要求
- 法律科普类内容需标注"仅供参考，不构成法律意见"
- 不得以律师身份承诺案件结果
- 不得引导用户通过非平台渠道交易
- 不得发布未经脱敏的真实案件信息

### 2.2 律师身份认证
- 抖音支持律师职业认证（蓝V）
- 认证需提供律师执业证
- 认证后可获得法律领域流量扶持

---

## 三、违禁词详细分类

### 3.1 绝对违禁词（直接封禁）
```
保证赢官司 / 包打赢 / 一定能赢
100%能要回来 / 保证能离婚
没有打不赢的官司
我认识XX法官 / 跟法院很熟
给我多少钱帮你搞定
```

### 3.2 高风险词（限流/警告）
```
最好的律师 / 最专业 / 第一
肯定没问题 / 你放心
对方必输 / 对方肯定理亏
我帮你搞定 / 包在我身上
这种案子我接了很多
```

### 3.3 引流类违禁词（限流）
```
加我微信 / 扫码加群
私聊发链接 / 点我主页
加V / VX / 威信
点击下方链接
淘宝/拼多多店铺名
```

### 3.4 诱导类违禁词（警告）
```
不转不是中国人 / 转发保平安
不看后悔一辈子 / 震惊所有人
速看马上删 / 内部消息
```

## 四、安全替代表达清单

| 违禁表达 | 安全替代 |
|----------|---------|
| 保证赢 | 根据现有证据和法律规定，胜诉可能性较大 |
| 一定能要回来 | 法律支持你的主张，可以通过合法途径追偿 |
| 包离 | 符合法定离婚条件的，法院会依法判决 |
| 我帮你搞定 | 我可以为你提供专业的法律分析和建议 |
| 最好的办法 | 比较稳妥的处理方式是…… |
| 对方必输 | 对方的法律依据存在明显漏洞 |
| 你放心 | 法律规定对你有利 |
| 肯定没问题 | 从法律角度分析，风险较低 |
| 加我微信 | 可以通过平台私信联系我 |
| 这种案子我接了很多 | 在这个领域我有丰富的实务经验 |
| 我认识法官 | 我对这个法院的审判实践比较了解 |

## 五、律师执业纪律（与直播相关）

### 5.1 《律师执业行为规范》相关条款
- 不得以不正当手段招揽业务
- 不得承诺案件结果
- 不得贬低同行
- 不得泄露当事人秘密
- 不得暗示与司法人员有特殊关系

### 5.2 《律师和律师事务所违法行为处罚办法》
- 以不正当手段招揽业务：警告/罚款/停止执业
- 承诺案件结果：警告/罚款
- 泄露当事人秘密：停止执业/吊销执照

## 六、直播事故案例

### 案例1：承诺结果导致投诉
某律师直播中说"这个案子交给我，保证能赢"，被当事人录音后投诉到律协，受到警告处分。

### 案例2：未脱敏导致泄露
某律师直播中详细描述了一个在办案件，虽然未提当事人姓名，但细节足够让当事人被识别，被投诉泄露秘密。

### 案例3：引流到微信被限流
某律师每场直播都在引导观众加微信，被抖音判定为"引流到站外"，账号被限流30天。

## 七、合规直播检查清单

### 直播前
- [ ] 脚本已扫描违禁词
- [ ] 案例已完全脱敏
- [ ] 未承诺任何案件结果
- [ ] 未贬低同行或法院
- [ ] 已准备免责声明话术

### 直播中
- [ ] 实时监控弹幕，避免敏感话题失控
- [ ] 回答问题时先声明"一般性分析，不针对具体案件"
- [ ] 遇到敏感问题及时转移话题
- [ ] 不引导加微信等站外联系方式

### 直播后
- [ ] 回看直播录像，检查是否有违规内容
- [ ] 整理高频问题，优化下次内容
- [ ] 私信咨询分类处理
