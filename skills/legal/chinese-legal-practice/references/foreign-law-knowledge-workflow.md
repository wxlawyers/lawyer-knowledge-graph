# 涉外法律知识沉淀工作流

## 适用场景
为特定法律领域创建全维度知识库沉淀，覆盖诉讼/非诉/法规/接待/文书/案例/实务。

## 工作流程

### 1. 创建文件夹结构
```python
base_path = "/Users/yuzhenghong/Library/Mobile Documents/iCloud~md~obsidian/Documents/法律知识库/{编号}-{领域}"
folders = ["01-诉讼流程", "02-非诉业务", "03-法律法规", "04-司法解释",
           "05-客户接待", "06-文书模板", "07-案例研究", "08-实务指南"]
```

### 2. MCP检索权威法规
```
mcp_yuandian_search_fagui(keyword="核心法律名称") → 获取法规列表
mcp_pkulaw_search_article(text="法条关键词") → 交叉验证
mcp_yuandian_get_fagui_detail(id=...) → 获取法规全文
```

### 3. 创建知识文档
每个模块一个独立 .md 文件，包含：
- 核心法律依据（法条+条文）
- 实务操作要点
- Pitfalls（避坑指南）
- 相关链接（Obsidian双向链接）

### 4. 交叉验证
检查所有文件的法律引用一致性，确保：
- 核心法律在每个模块都有引用
- 法条编号准确
- 数据来源标注

### 5. 创建索引README
汇总所有模块的索引文件，包含：
- 知识库结构树
- 核心法律速查表
- 业务领域速查表

## Obsidian路径
`法律知识库/11-涉外法律/` 下8个子目录+README.md

## 质量标准
- 数据来源：元典智库 + 北大法宝双源交叉验证
- 法律引用：法条全称+条号+施行日期
- 格式规范：表格优先、结论先行
