# 证据四性验证标准

## 一、真实性验证

### 书证
- 是否原件？复印件是否有原件核对？
- 是否有涂改、伪造痕迹？
- 签名/盖章是否真实？
- 形成时间是否明确？

### 电子数据
- 原始存储介质是否保存？
- 是否有篡改可能？
- 公证保全或区块链存证？
- 提取过程是否有见证人？

### 证人证言
- 证人与当事人关系？
- 证人是否具备作证能力？
- 证言是否前后一致？
- 是否有其他证据佐证？

## 二、合法性验证

### 取证方式
- 是否侵犯他人合法权益？
- 是否违反法律禁止性规定？
- 是否严重违背公序良俗？

### 常见非法证据
- 偷录偷拍（侵犯隐私权）
- 非法侵入他人住宅获取
- 通过暴力、威胁手段获取
- 严重违反法定程序获取

## 三、关联性验证

### 判断标准
- 证据与待证事实是否有逻辑联系？
- 证据能否证明或反驳某项事实？
- 证据与其他证据是否矛盾？

### 常见无关联证据
- 与案件无关的背景信息
- 当事人的品格证据（一般情况）
- 其他案件的证据

## 四、充分性验证

### 证据链完整性
- 是否形成完整证据链？
- 是否有缺失环节？
- 能否排除合理怀疑？

### 证明标准
- 民事：高度盖然性（优势证据）
- 刑事：排除合理怀疑
- 行政：清楚且有说服力

## 六、质证要点

### 对方证据质证
1. 真实性异议：是否原件？有无篡改？
2. 合法性异议：取证方式是否合法？
3. 关联性异议：与本案是否相关？
4. 证明目的异议：能否证明对方主张？

### 我方证据补强
1. 是否有其他证据佐证？
2. 是否需要申请鉴定？
3. 是否需要申请法院调查取证？
4. 是否需要申请证人出庭？
