# 法律案例搜索源指南

面向中国诉讼律师的案例搜索多源检索指南，覆盖四大权威案例来源。

---

## 一、中国裁判文书网（wenshu.court.gov.cn）

### 合法使用方式

**推荐方式**：手动在浏览器中搜索，不要使用自动化爬取工具。

裁判文书网有严格的反爬机制和使用条款，自动化爬取可能违反《网站使用协议》及相关法律法规。

### 手动搜索技巧

在裁判文书网搜索框中输入以下关键词组合：

| 搜索目标 | 关键词组合 |
|----------|------------|
| 建设工程施工合同 | `"建设工程施工合同" AND "江苏省高级人民法院"` |
| 实际施工人 | `"实际施工人" AND "工程价款" AND "江苏"` |
| 优先受偿权 | `"建设工程价款优先受偿权" AND "江苏"` |
| 工程质量 | `"建设工程" AND "质量缺陷" AND "江苏"` |
| 挂靠 | `"借用资质" OR "挂靠" AND "建设工程" AND "江苏"` |

### 筛选条件

- **法院层级**：高级人民法院 / 中级人民法院
- **地域**：江苏省
- **案件类型**：民事案件
- **案由**：建设工程合同纠纷
- **裁判年份**：近3-5年

## 三、元典智库（MCP工具）

### 可用工具

| 工具 | 用途 | 搜索建议 |
|------|------|----------|
| `mcp_yuandian_search_fagui` | 法规检索 | 搜索司法文件、审判指导意见效果好 |
| `mcp_yuandian_search_fatiao` | 法条检索 | 精确检索法条内容 |
| `mcp_yuandian_search_qwal` | 案例检索 | ⚠️ 精度差，建设工程关键词常返回不相关案例 |
| `mcp_yuandian_get_fagui_detail` | 法规详情 | 获取法规全文 |

### 案例检索优化建议

元典智库的案例检索（search_qwal）对建设工程关键词匹配精度差，建议：
1. 使用更精确的案号检索（如果已知）
2. 结合法规检索（search_fagui）获取司法文件
3. 用浏览器访问中国裁判文书网或北大法宝补充案例

### 可用工具

| 工具 | 用途 |
|------|------|
| `mcp_pkulaw_case_search_search_case` | 案例智能检索 |
| `mcp_pkulaw_case_get_case_list` | 案例列表查询 |
| `mcp_pkulaw_law_get_law_list` | 法规列表查询 |
| `mcp_pkulaw_search_article` | 法条语义检索 |

### 搜索建议

```

mcp_pkulaw_case_search_search_case(text="建设工程施工合同 江苏省高级人民法院")

# 搜索实际施工人案例
mcp_pkulaw_case_search_search_case(text="实际施工人 工程价款 江苏")

# 搜索优先受偿权案例
mcp_pkulaw_case_search_search_case(text="建设工程价款优先受偿权 江苏")
```

## 五、多源检索策略

### 第一轮：MCP工具检索

1. 元典智库 `search_fagui` → 获取司法文件全文
2. 北大法宝 `case_search_search_case` → 搜索相关案例

### 第二轮：浏览器检索

1. 江苏法院网搜索 → 获取江苏本地典型案例
2. 中国裁判文书网 → 补充裁判文书

### 第三轮：交叉验证

1. 对比多源检索结果
2. 核实案例案号、法院、日期
3. 提取裁判要旨

## 六、常见错误

### 1. 元典智库案例检索精度差
**问题**：建设工程关键词常返回不相关的知识产权、股权转让案例
**解决**：用法规检索替代，或用浏览器访问裁判文书网

### 2. 中国裁判文书网反爬
**问题**：自动化爬取违反网站使用协议，可能触犯法律
**解决**：仅使用手动搜索方式，或使用北大法宝/元典智库等合法数据服务

### 3. 北大法宝MCP服务不稳定
**问题**：可能返回 "MCP server not connected" 错误
**解决**：重试或使用元典智库替代
