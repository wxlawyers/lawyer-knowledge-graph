# 并购重组（M&A）知识技能

## 一、交易类型

| 类型 | 定义 | 税务处理 | 法律依据 |
|------|------|---------|---------|
| 股权收购 | 收购目标公司股权 | 印花税+所得税 | 公司法第71条 |
| 资产收购 | 收购目标公司资产 | 增值税+所得税+契税 | 合同法 |
| 合并 | 两个公司合为一个 | 特殊性税务处理 | 公司法第172条 |
| 分立 | 一个公司分为多个 | 特殊性税务处理 | 公司法第175条 |

## 二、尽职调查清单

### 2.1 法律尽调
```
□ 公司章程、股东协议
□ 营业执照、工商登记
□ 股东名册、股权结构
□ 重大合同（前10大客户/供应商）
□ 诉讼/仲裁案件清单

□ 劳动合同、社保缴纳
□ 环保批文、资质许可
□ 关联交易清单
□ 政府批文、行政许可
```

### 2.2 财务尽调
```
□ 三年一期审计报告
□ 管理层报表
□ 税务申报表
□ 银行贷款合同
□ 应收/应付账款账龄分析

□ 关联方资金往来
```

## 三、对赌协议

### 3.1 对赌类型

| 类型 | 内容 | 风险 |
|------|------|------|
| 业绩对赌 | 净利润不低于X万元 | 业绩不达标需补偿 |
| 上市对赌 | X年内完成IPO | 未上市需回购 |
| 用户/收入对赌 | 用户数/收入达到X | 数据造假风险 |

### 3.2 补偿方式

```
现金补偿：实际利润/承诺利润 × 投资总额
股权补偿：(1 - 实际利润/承诺利润) × 原持股比例
回购补偿：投资总额 × (1+年化收益率)^n
```

### 3.3 对赌效力

**有效情形**：
- 投资方与目标公司股东对赌：有效
- 投资方与目标公司对赌：需符合公司法规定

**无效情形**：
- 损害公司债权人利益
- 违反公司资本维持原则

**法律依据**：
- 《九民纪要》第5条：对赌协议效力认定

## 四、交易条款

### 4.1 反稀释条款

```
完全棘轮：新价格 < 原价格 → 调整后价格 = 新价格
加权平均：调整后价格 = (原价格 × 原股数 + 新价格 × 新股数) / (原股数 + 新股数)
```

### 4.2 拖拽权/随售权

```
拖拽权（Drag-Along）：大股东出售股权时，可强制小股东一起出售
随售权（Tag-Along）：大股东出售股权时，小股东有权按同等条件一起出售
```

### 4.3 优先购买权

```
公司法第71条：股东向股东以外的人转让股权，应当经其他股东过半数同意
其他股东在同等条件下有优先购买权
```

## 五、税务处理

### 5.1 一般性税务处理

```
股权收购：
- 被收购方：确认股权转让所得/损失
- 收购方：以公允价值为计税基础

资产收购：
- 被收购方：确认资产转让所得/损失
- 收购方：以公允价值为计税基础
```

### 5.2 特殊性税务处理

**条件**：
- 收购比例≥50%
- 支付对价中股权支付比例≥85%
- 12个月内不改变被收购资产原有经营活动

**效果**：
- 被收购方：暂不确认所得/损失
- 收购方：以原有计税基础确定

## 六、法律依据

- 《公司法》第71条（股权转让）
- 《公司法》第172-176条（合并分立）
- 《证券法》（上市公司收购）
- 《上市公司重大资产重组管理办法》
- 《九民纪要》第5条（对赌协议）
