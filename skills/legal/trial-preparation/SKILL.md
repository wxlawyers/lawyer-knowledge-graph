# 庭前准备技能

面向中国诉讼律师的庭前准备框架，提供从质证要点到庭审策略的全流程支持。

## 适用场景

- 质证要点梳理
- 发问提纲设计
- 对方抗辩预判
- 庭审策略制定
- 庭前模拟演练

## 执行步骤

### 步骤一：案件梳理

#### 案件事实梳理
1. **时间线整理**
   - 按时间顺序排列事件
   - 标注关键时间节点
   - 确认事实真实性

2. **争议焦点识别**
   - 事实争议焦点
   - 法律争议焦点
   - 程序争议焦点

3. **证据梳理**
   - 我方证据清单
   - 对方证据清单
   - 证据强弱分析

#### 法律关系分析
1. **请求权基础**
   - 请求权主体
   - 请求权客体
   - 请求权内容
   - 法律依据

2. **抗辩权分析**
   - 诉讼时效抗辩
   - 合同效力抗辩
   - 履行抗辩权
   - 抵销权

### 步骤二：质证要点梳理

#### 对方证据质证

**1. 真实性异议**
- 是否原件？
- 有无篡改痕迹？
- 签名/盖章是否真实？
- 形成时间是否明确？

**2. 合法性异议**
- 取证方式是否合法？
- 是否侵犯他人权益？
- 是否违反法定程序？

**3. 关联性异议**
- 与本案是否相关？
- 能否证明对方主张？
- 是否有其他证据佐证？

**4. 证明目的异议**
- 能否证明对方主张？
- 是否有其他解释？
- 是否与其他证据矛盾？

#### 我方证据补强

**1. 证据补强**
- 是否有其他证据佐证？
- 是否需要申请鉴定？
- 是否需要申请法院调查取证？

**2. 证据说明**
- 证据来源说明
- 证据形成过程
- 证据证明力说明

### 步骤三：发问提纲设计

#### 发问对象
1. **对方当事人**
   - 事实确认
   - 细节追问
   - 矛盾揭示

2. **证人**
   - 证言真实性
   - 证人可信度
   - 细节确认

3. **鉴定人**
   - 鉴定依据
   - 鉴定过程
   - 鉴定结论

#### 发问策略
1. **开放式问题**
   - 了解事实全貌
   - 获取更多信息
   - 引导对方陈述

2. **封闭式问题**
   - 确认具体事实
   - 揭示矛盾
   - 限制对方回答

3. **诱导式问题**
   - 引导对方承认
   - 揭示矛盾
   - 强化我方观点

#### 发问提纲模板
```
发问提纲

一、对方当事人发问
1. （开放式）请描述一下事件经过？
2. （封闭式）你是否在XX时间XX地点？
3. （诱导式）你当时是否承认了XX事实？

二、证人发问
1. （开放式）请描述一下你看到的情况？
2. （封闭式）你是否亲眼看到了XX？
3. （诱导式）你是否记得当时XX情况？

三、鉴定人发问
1. （开放式）鉴定依据是什么？
2. （封闭式）鉴定过程是否符合规范？
3. （诱导式）鉴定结论是否排除了其他可能？
```

### 步骤四：对方抗辩预判

#### 常见抗辩类型
1. **诉讼时效抗辩**
   - 是否超过诉讼时效？
   - 是否有中断事由？
   - 是否有中止事由？

2. **合同效力抗辩**
   - 合同是否有效？
   - 是否有无效事由？
   - 是否有可撤销事由？

3. **履行抗辩权**
   - 同时履行抗辩权
   - 先履行抗辩权
   - 不安抗辩权

4. **抵销权**
   - 是否符合抵销条件？
   - 抵销金额是否准确？
   - 抵销是否合法？

#### 抗辩应对策略
1. **诉讼时效抗辩应对**
   - 提供中断证据
   - 提供中止证据
   - 主张对方放弃时效利益

2. **合同效力抗辩应对**
   - 提供合同有效证据
   - 提供合同履行证据
   - 主张合同有效

3. **履行抗辩权应对**
   - 提供已履行证据
   - 提供对方违约证据
   - 主张抗辩权不成立

4. **抵销权应对**
   - 提供债权不存在证据
   - 提供抵销条件不成就证据
   - 主张抵销不成立

### 步骤五：庭审策略制定

#### 庭审目标
1. **主要目标**
   - 争取胜诉
   - 争取有利判决
   - 争取调解

2. **次要目标**
   - 争取部分支持
   - 争取减少损失
   - 争取时间

#### 庭审策略
1. **进攻策略**
   - 主动出击
   - 揭示对方矛盾
   - 强化我方观点

2. **防守策略**
   - 稳扎稳打
   - 化解对方攻击
   - 维护我方利益

3. **调解策略**
   - 争取调解
   - 降低期望
   - 争取最优方案

#### 庭审提纲模板
```
庭审提纲

一、开场陈述
1. 案件概述
2. 争议焦点
3. 我方观点

二、举证质证
1. 我方举证
   - 证据1：XXX
   - 证据2：XXX
2. 对方质证
   - 质证意见1：XXX
   - 质证意见2：XXX
3. 我方质证
   - 质证意见1：XXX
   - 质证意见2：XXX

三、法庭辩论
1. 争议焦点1：XXX
   - 我方观点：XXX
   - 对方观点：XXX
   - 反驳：XXX
2. 争议焦点2：XXX
   - 我方观点：XXX
   - 对方观点：XXX
   - 反驳：XXX

四、最后陈述
1. 总结观点
2. 明确请求
3. 简洁有力
```

### 步骤六：庭前模拟演练

#### 模拟内容
1. **开场陈述模拟**
   - 练习陈述
   - 控制时间
   - 调整语速

2. **举证质证模拟**
   - 练习举证
   - 练习质证
   - 调整策略

3. **法庭辩论模拟**
   - 练习辩论
   - 练习反驳
   - 调整策略

4. **最后陈述模拟**
   - 练习陈述
   - 控制时间
   - 调整语速

#### 模拟方法
1. **自我模拟**
   - 独自练习
   - 录音录像
   - 回放改进

2. **同事模拟**
   - 同事扮演对方
   - 模拟庭审
   - 提出改进意见

3. **专家模拟**
   - 专家指导
   - 模拟庭审
   - 提出专业意见

## 质量标准

### 庭前准备质量
1. **全面性** — 是否覆盖所有争议焦点
2. **针对性** — 是否有针对性应对策略
3. **可操作性** — 是否具有可操作性
4. **灵活性** — 是否具有应变能力

### 庭审策略质量
1. **目标明确** — 庭审目标是否明确
2. **策略合理** — 庭审策略是否合理
3. **应变能力** — 是否具有应变能力
4. **调解意愿** — 是否有调解意愿

## 安全要求

- 案件信息保密
- 庭审策略保密
- 不对外泄露案件信息
