# 司法会计鉴定知识技能

## 一、司法会计鉴定

### 1.1 鉴定类型

| 类型 | 内容 | 适用场景 |
|------|------|---------|
| 财务状况鉴定 | 资产、负债、所有者权益 | 破产、清算 |
| 经营成果鉴定 | 收入、成本、利润 | 合同纠纷、利润分配 |
| 现金流量鉴定 | 经营、投资、筹资现金流 | 偿债能力争议 |
| 损失金额鉴定 | 直接损失、间接损失 | 侵权纠纷、违约纠纷 |
| 财务往来鉴定 | 资金流向、往来账款 | 民间借贷、合伙纠纷 |

### 1.2 鉴定程序

```
1. 委托：法院/仲裁机构/当事人委托
2. 受理：鉴定机构审查材料
3. 鉴定：查阅账簿、凭证、报表
4. 出具鉴定意见
5. 质证：当事人对鉴定意见质证
6. 采信：法院决定是否采信
```

### 1.3 鉴定机构

- 会计师事务所
- 司法鉴定机构（具有司法会计鉴定资质）

## 二、损失计算

### 2.1 直接损失

```
直接损失 = 现有财产的减少

计算方法：
- 财产灭失：市场价值
- 财产损坏：修复费用或贬值金额
- 合同违约：实际损失金额
```

### 2.2 间接损失

```
间接损失 = 可得利益的丧失

计算方法：
- 利润损失：正常经营利润 - 实际利润
- 机会损失：合理预期收益
- 增值损失：财产增值部分
```

### 2.3 典型案例计算

**合同违约损失**：
```
损失 = 直接损失 + 可得利益损失

直接损失：已付款项、费用支出
可得利益损失：合同履行后可以获得的利益
限制：不得超过违约方订立合同时预见到或者应当预见到的损失
```

**侵权损失**：
```
损失 = 实际损失 或 侵权获利

实际损失：因侵权行为导致的财产减少
侵权获利：侵权人因侵权行为获得的利益
```

## 三、利润分配争议

### 3.1 分配条件

- 公司有可分配利润
- 股东会作出分配决议
- 符合公司章程规定

### 3.2 分配规则

```
法定顺序：
1. 弥补以前年度亏损
2. 提取法定公积金（10%）
3. 提取任意公积金
4. 向股东分配

分配比例：
- 有限责任公司：按实缴出资比例
- 股份有限公司：按持股比例
- 公司章程另有规定的除外
```

### 3.3 常见争议

**大股东不分红**：
- 救济途径：请求公司回购股权（公司法第74条）
- 条件：公司连续5年盈利且连续5年不分红

**利润分配决议无效**：
- 违反法定顺序
- 损害债权人利益
- 未经股东会决议

## 四、审计报告审查

### 4.1 审查要点

```
1. 审计机构资质
   - 是否具有执业资格
   - 是否与被审计单位存在利害关系

2. 审计程序
   - 是否执行了必要的审计程序
   - 审计范围是否完整

3. 审计证据
   - 证据是否充分、适当
   - 证据是否真实、合法

4. 审计结论
   - 结论是否有充分依据
   - 结论是否与证据一致
```

### 4.2 质证要点

**对审计报告的质证**：
```
1. 审计机构独立性
   - 是否与对方存在利害关系
   - 是否收取了不合理费用

2. 审计程序合规性
   - 是否执行了必要的审计程序
   - 审计范围是否受限

3. 审计证据充分性
   - 证据来源是否可靠
   - 证据是否足以支持结论

4. 审计结论合理性
   - 结论是否有充分依据
   - 是否存在逻辑矛盾
```

### 4.3 常见问题

| 问题 | 质证角度 | 法律后果 |
|------|---------|---------|
| 审计范围受限 | 未获取充分证据 | 审计意见不可靠 |
| 审计程序缺失 | 未执行必要程序 | 审计意见不可靠 |
| 关联方交易 | 未充分披露 | 可能存在利益输送 |
| 会计政策变更 | 变更理由不充分 | 可能操纵利润 |

## 五、财务证据质证

### 5.1 账簿凭证质证

```
1. 真实性
   - 是否原始凭证
   - 是否有篡改痕迹
   - 签章是否真实

2. 合法性
   - 是否符合会计准则
   - 是否经过适当审批

3. 关联性
   - 与待证事实是否相关
   - 能否证明待证事实
```

### 5.2 银行流水质证

```
1. 完整性
   - 是否提供全部流水
   - 是否有遗漏账户

2. 真实性
   - 是否银行原件
   - 金额是否准确

3. 关联性
   - 资金流向是否与案件相关
   - 是否存在其他合理解释
```

### 5.3 审计报告质证

```
1. 独立性质证
   - 审计机构是否独立
   - 是否存在利益冲突

2. 程序性质证
   - 是否执行了必要程序
   - 审计范围是否受限

3. 证据充分性质证
   - 证据是否充分、适当
   - 是否存在重大遗漏

4. 结论合理性质证
   - 结论是否有充分依据
   - 是否存在逻辑矛盾
```

## 六、速查表

### 6.1 损失计算速查

```
直接损失 = 现有财产减少
间接损失 = 可得利益丧失
总损失 = 直接损失 + 间接损失

限制：可预见性规则（违约方订立合同时预见到的损失）
```

### 6.2 利润分配速查

```
可分配利润 = 净利润 - 以前年度亏损 - 法定公积金 - 任意公积金
分配比例：按出资比例（章程另有规定除外）
救济途径：连续5年不分红 → 请求公司回购股权
```

### 6.3 审计报告质证速查

```
质证角度：
1. 独立性（是否存在利害关系）
2. 程序性（是否执行必要程序）
3. 充分性（证据是否充分适当）
4. 合理性（结论是否有充分依据）
```
