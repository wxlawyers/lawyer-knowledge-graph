# Excalidraw Patterns for Obsidian Knowledge Base

## 知识卡片配图模式

### 模式1：思维导图（法规/知识点概览）
- 中心节点：主题（蓝色 #a5d8ff）
- 一级分支：核心要点（绿色 #b2f2bb）
- 二级分支：细节说明（黄色 #fff3bf）
- 适用于：法规解读、知识点总结

### 模式2：流程图（实务操作流程）
- 起点节点：触发条件（绿色 #b2f2bb）
- 处理节点：操作步骤（蓝色 #a5d8ff）
- 判断节点：条件分支（黄色 #fff3bf）
- 终点节点：结果（红色 #ffc9c9 用于风险，绿色用于成功）
- 适用于：诉讼流程、合规检查、操作指南

### 模式3：关系图（案例分析）
- 当事人节点：原告/被告（蓝色/橙色）
- 争议焦点：中心节点（紫色 #d0bfff）
- 裁判要旨：结论节点（绿色）
- 适用于：案例卡片、裁判规则

### 模式4：时间线（法规生效/案件进展）
- 横向排列，时间从左到右
- 节点颜色区分：已生效=绿色，待生效=橙色，失效=红色
- 适用于：法规过渡期、案件时间线

## Obsidian集成规范

### 文件命名
- 思维导图：`{主题}-思维导图.excalidraw`
- 流程图：`{主题}-流程图.excalidraw`
- 关系图：`{主题}-关系图.excalidraw`
- 时间线：`{主题}-时间线.excalidraw`

### 存放位置
- 与对应的.md文件同一目录
- 或统一存放在 `Excalidraw/` 目录，在.md中用相对路径引用

### 嵌入方式（Obsidian）
```markdown
![[文件名.excalidraw]]
```

## 颜色速查表

| 用途 | 颜色 | Hex |
|------|------|-----|
| 主题/核心 | 蓝色 | #a5d8ff |
| 成功/输出 | 绿色 | #b2f2bb |
| 警告/外部 | 橙色 | #ffd8a8 |
| 特殊/程序 | 紫色 | #d0bfff |
| 错误/风险 | 红色 | #ffc9c9 |
| 注释/决策 | 黄色 | #fff3bf |
| 数据/存储 | 青色 | #c3fae8 |
