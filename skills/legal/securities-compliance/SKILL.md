# 证券合规知识技能

## 一、信息披露

### 1.1 披露义务人

- 发行人、上市公司
- 董事、监事、高级管理人员
- 控股股东、实际控制人
- 保荐人、承销的证券公司
- 证券服务机构

### 1.2 披露类型

| 类型 | 内容 | 时限 |
|------|------|------|
| 定期报告 | 年报、半年报、季报 | 法定期限内 |
| 临时报告 | 重大事件 | 立即 |
| 招股说明书 | IPO信息 | 发行前 |

### 1.3 虚假陈述

**构成要件**：
- 信息披露义务人实施了虚假陈述行为
- 投资者在虚假陈述实施日后、揭露日前买入证券
- 投资者在揭露日后卖出或持有证券产生亏损
- 虚假陈述与投资者损失之间存在因果关系

**法律后果**：
- 行政责任：警告、罚款、市场禁入
- 民事责任：赔偿投资者损失
- 刑事责任：违规披露、不披露重要信息罪

## 二、内幕交易

### 2.1 内幕信息

**法定内幕信息**：
- 公司经营方针和经营范围的重大变化
- 公司的重大投资行为和重大的购置财产的决定
- 公司订立重要合同，可能对公司的资产、负债、权益和经营成果产生重要影响
- 公司发生重大亏损或者重大损失
- 公司生产经营的外部条件发生的重大变化
- 公司的董事、1/3以上监事或者经理发生变动
- 持有公司5%以上股份的股东或者实际控制人，其持有股份或者控制公司的情况发生较大变化
- 公司减资、合并、分立、解散及申请破产的决定
- 涉及公司的重大诉讼，股东大会、董事会决议被依法撤销或者宣告无效
- 公司涉嫌犯罪被司法机关立案调查
- 公司分配股利或者增资的计划
- 公司股权结构的重大变化
- 公司债务担保的重大变更
- 公司营业用主要资产的抵押、出售或者报废一次超过该资产的30%
- 公司的董事、监事、高级管理人员的行为可能依法承担重大损害赔偿责任
- 上市公司收购的有关方案
- 国务院证券监督管理机构认定的对证券交易价格有显著影响的其他重要信息

### 2.2 内幕交易行为

- 买入或者卖出该证券
- 泄露该信息
- 建议他人买卖该证券

### 2.3 法律后果

**行政责任**：
- 没收违法所得
- 处以违法所得1-10倍罚款
- 没有违法所得或违法所得不足50万元的，处以50-500万元罚款

**刑事责任**：
- 《刑法》第180条：内幕交易罪
- 情节严重：5年以下有期徒刑或者拘役
- 情节特别严重：5年以上10年以下有期徒刑

## 三、操纵市场

### 3.1 操纵行为

- 单独或者通过合谋，集中资金优势、持股优势或者利用信息优势联合或者连续买卖
- 与他人串通，以事先约定的时间、价格和方式相互进行证券交易
- 在自己实际控制的账户之间进行证券交易
- 不以成交为目的，频繁或者大量申报并撤销申报
- 利用虚假或者不确定的重大信息，诱导投资者进行证券交易
- 对证券、发行人公开作出评价、预测或者投资建议，并进行反向证券交易
- 利用其他手段操纵证券市场

### 3.2 法律后果

**行政责任**：
- 没收违法所得
- 处以违法所得1-10倍罚款
- 没有违法所得或违法所得不足100万元的，处以100-1000万元罚款

**刑事责任**：
- 《刑法》第182条：操纵证券、期货市场罪
- 情节严重：5年以下有期徒刑或者拘役
- 情节特别严重：5年以上10年以下有期徒刑

## 四、投资者索赔

### 4.1 索赔条件

- 虚假陈述实施日后买入
- 虚假陈述揭露日前持有
- 揭露日后卖出或继续持有产生亏损
- 买入与虚假陈述之间存在交易因果关系

### 4.2 索赔流程

```
1. 确认虚假陈述行为（行政处罚决定/刑事判决）
2. 计算投资差额损失
3. 向有管辖权的法院提起诉讼
4. 法院审理（代表人诉讼/示范判决）
5. 获得赔偿
```

### 4.3 损失计算

```
投资差额损失 = (买入均价 - 卖出均价或基准价) × 证券数量

基准价：虚假陈述揭露日起至基准日期间，每个交易日收盘价的平均价
```

## 五、上市公司治理

### 5.1 三会制度

**股东大会**：
- 最高权力机构
- 决定公司经营方针和投资计划
- 选举和更换董事、监事

**董事会**：
- 执行机构
- 制定公司经营计划和投资方案
- 制定公司年度财务预算方案

**监事会**：
- 监督机构
- 检查公司财务
- 对董事、高级管理人员执行公司职务的行为进行监督

### 5.2 独立董事

- 独立性要求：与公司不存在可能影响独立客观判断的关系
- 职责：对关联交易、高管聘任、薪酬等发表独立意见
- 特别职权：提议召开董事会、提议召开股东大会、独立聘请外部审计机构

## 六、速查表

### 6.1 证券违法行为处罚

| 违法行为 | 行政罚款 | 刑事责任 |
|----------|---------|---------|
| 虚假陈述 | 100-1000万 | 3-10年 |
| 内幕交易 | 违法所得1-10倍 | 5-10年 |
| 操纵市场 | 违法所得1-10倍 | 5-10年 |
| 欺诈发行 | 非法募集资金2%-10% | 5-15年 |

### 6.2 投资者索赔条件

```
虚假陈述：
✓ 实施日后买入
✓ 揭露日前持有
✓ 揭露日后卖出/持有亏损
✓ 交易因果关系

内幕交易/操纵市场：
✓ 行为期间买入/卖出
✓ 产生损失
✓ 损失与违法行为有因果关系
```
