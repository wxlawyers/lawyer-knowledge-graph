# 破产清算知识技能

## 一、破产申请

### 1.1 申请条件

**企业法人**：
- 不能清偿到期债务
- 资产不足以清偿全部债务（资不抵债）
- 明显缺乏清偿能力

**法律依据**：《企业破产法》第2条

### 1.2 申请人

| 申请人 | 条件 | 法律依据 |
|--------|------|---------|
| 债务人 | 不能清偿到期债务 | 第7条第1款 |
| 债权人 | 债务人不能清偿到期债务 | 第7条第2款 |
| 清算组 | 解散后资不抵债 | 第7条第3款 |

### 1.3 管辖法院

- 地域管辖：债务人住所地人民法院
- 级别管辖：基层人民法院管辖一般破产案件

## 二、债权申报

### 2.1 申报期限

- 法院受理破产申请后，确定债权申报期限
- 最短30日，最长3个月
- 自人民法院发布受理破产申请公告之日起计算

### 2.2 申报材料

```
□ 债权申报书
□ 债权人身份证明
□ 债权凭证（合同、判决书等）
□ 债权金额计算依据
□ 担保物权证明（如有）
□ 委托书（代理人申报）
```

### 2.3 债权审查

**管理人审查**：
- 形式审查：申报材料是否齐全
- 实质审查：债权是否成立、金额是否正确

**债权人会议核查**：
- 对管理人编制的债权表进行核查
- 债务人、债权人对债权有异议的，可以向受理破产申请的人民法院提起诉讼

## 三、破产财产

### 3.1 财产范围

**属于破产财产**：
- 破产申请受理时属于债务人的全部财产
- 破产申请受理后至破产程序终结前债务人取得的财产

**不属于破产财产**：
- 债务人基于仓储、保管、承揽、代销、借用、寄存、租赁等合同占有、使用的他人财产
- 担保物灭失后产生的保险金、补偿金、赔偿金等代位物
- 所有权保留买卖中尚未取得所有权的财产

### 3.2 财产分配顺序

```
1. 破产费用和共益债务
2. 职工债权（工资、社保、补偿金）
3. 税款债权
4. 普通破产债权

同一顺序不足清偿的，按比例分配
```

## 四、重整程序

### 4.1 重整申请

- 债务人或出资额占债务人注册资本1/10以上的出资人
- 在人民法院受理破产申请后、宣告债务人破产前提出

### 4.2 重整计划内容

```
1. 债务人的经营方案
2. 债权分类
3. 债权调整方案
4. 债权受偿方案
5. 重整计划的执行期限
6. 重整计划执行的监督期限
7. 有利于债务人重整的其他方案
```

### 4.3 表决规则

- 出席会议的同一表决组的债权人过半数同意
- 其所代表的债权额占该组债权总额的2/3以上

## 五、管理人职责

### 5.1 法定职责

```
1. 接管债务人的财产、印章和账簿、文书等资料
2. 调查债务人财产状况，制作财产状况报告
3. 决定债务人的内部管理事务
4. 决定债务人的日常开支和其他必要开支
5. 在第一次债权人会议召开之前，决定继续或者停止债务人的营业
6. 管理和处分债务人的财产
7. 代表债务人参加诉讼、仲裁或者其他法律程序
8. 提议召开债权人会议
9. 人民法院认为管理人应当履行的其他职责
```

### 5.2 管理人责任

- 忠实义务：不得利用职权为自己牟取私利
- 勤勉义务：尽到善良管理人的注意义务
- 赔偿责任：因故意或重大过失造成损失的，应当承担赔偿责任

## 六、破产撤销权

### 6.1 可撤销行为

**受理前1年内**：
- 无偿转让财产
- 以明显不合理的价格进行交易
- 对没有财产担保的债务提供财产担保
- 对未到期的债务提前清偿
- 放弃债权

**受理前6个月内**：
- 债务人已具备破产原因，仍对个别债权人清偿

### 6.2 无效行为

- 为逃避债务而隐匿、转移财产
- 虚构债务或者承认不真实的债务

## 七、速查表

### 7.1 破产程序选择

```
资不抵债 + 有重整价值 → 重整程序
资不抵债 + 无重整价值 → 清算程序
有清偿能力 + 需要债务调整 → 和解程序
```

### 7.2 债权清偿顺序

```
1. 破产费用（诉讼费、管理人报酬等）
2. 共益债务（为债权人利益产生的债务）
3. 职工债权（工资、社保、经济补偿金）
4. 税款债权
5. 普通破产债权
```

### 7.3 关键时间节点

```
申请受理：法院15日内裁定（特殊情况延长15日）
债权申报：公告之日起30日-3个月
第一次债权人会议：债权申报期限届满15日内
重整计划执行：最长不超过6个月（可延长3个月）
```
