# 法律文书撰写技能

面向中国诉讼律师的法律文书撰写框架，提供各类诉讼文书的撰写规范和模板。

## 适用场景

- 起诉状撰写
- 答辩状撰写
- 代理词撰写
- 辩护词撰写
- 法律意见书撰写

## 执行步骤

### 步骤一：确定文书类型

#### 诉讼文书
1. **起诉状**
   - 民事起诉状
   - 行政起诉状
   - 刑事自诉状

2. **答辩状**
   - 民事答辩状
   - 行政答辩状

3. **上诉状**
   - 民事上诉状
   - 行政上诉状
   - 刑事上诉状

4. **申请书**
   - 财产保全申请书
   - 证据保全申请书
   - 调查取证申请书
   - 鉴定申请书

#### 非诉文书
1. **代理词**
   - 原告代理词
   - 被告代理词
   - 第三人代理词

2. **辩护词**
   - 无罪辩护词
   - 罪轻辩护词
   - 量刑辩护词

3. **法律意见书**
   - 合同审查意见书
   - 尽职调查报告
   - 法律风险评估

### 步骤二：起诉状撰写

#### 基本格式
```
民事起诉状

原告：XXX，性别，出生日期，民族，住所地，身份证号，联系电话。

被告：XXX，性别，出生日期，民族，住所地，身份证号，联系电话。

诉讼请求：
1. 判令被告XXX
2. 判令被告赔偿XXX元
3. 本案诉讼费由被告承担

事实与理由：
（详细叙述案件事实和法律依据）

证据和证据来源，证人姓名和住所：
（列明证据清单）

此致
XXX人民法院

具状人：XXX
年 月 日
```

#### 撰写要点
1. **当事人信息**
   - 姓名/名称准确
   - 身份信息完整
   - 联系方式明确

2. **诉讼请求**
   - 明确具体
   - 可执行
   - 有法律依据

3. **事实与理由**
   - 时间顺序清晰
   - 事实陈述客观
   - 法律依据明确

4. **证据清单**
   - 编号连续
   - 证明目的明确
   - 页码准确

### 步骤三：答辩状撰写

#### 基本格式
```
民事答辩状

答辩人：XXX，性别，出生日期，民族，住所地，身份证号，联系电话。

因原告XXX诉答辩人XXX纠纷一案，答辩人提出如下答辩意见：

一、案件事实
（对原告主张的事实进行回应）

二、答辩意见
（提出答辩理由和法律依据）

三、诉讼请求
（提出反诉或反驳请求）

此致
XXX人民法院

答辩人：XXX
年 月 日
```

#### 撰写要点
1. **事实回应**
   - 逐项回应原告主张
   - 承认或否认事实
   - 提供相反证据

2. **答辩理由**
   - 法律依据明确
   - 逻辑清晰
   - 有针对性

3. **反驳请求**
   - 请求驳回原告诉讼请求
   - 提出反诉（如有）

### 步骤四：代理词撰写

#### 基本格式
```
代理词

审判长、审判员：

XXX律师事务所接受本案原告/被告XXX的委托，指派本律师担任其诉讼代理人。现根据本案事实和法律，提出如下代理意见：

一、案件事实
（概述案件基本事实）

二、代理意见
（提出代理观点和法律依据）

三、结论
（总结代理意见，提出诉讼请求）

以上代理意见，请法庭予以考虑。

代理人：XXX
年 月 日
```

#### 撰写要点
1. **事实概述**
   - 客观全面
   - 重点突出
   - 逻辑清晰

2. **代理意见**
   - 观点明确
   - 法律依据充分
   - 论证有力

3. **结论**
   - 总结观点
   - 明确请求
   - 简洁有力

### 步骤五：辩护词撰写

#### 基本格式
```
辩护词

审判长、审判员：

XXX律师事务所接受本案被告人XXX的委托，指派本律师担任其辩护人。现根据本案事实和法律，提出如下辩护意见：

一、案件事实
（概述案件基本事实）

二、辩护意见
（提出辩护观点和法律依据）

三、结论
（总结辩护意见，提出辩护请求）

以上辩护意见，请法庭予以考虑。

辩护人：XXX
年 月 日
```

#### 辩护类型
1. **无罪辩护**
   - 事实不清、证据不足
   - 不构成犯罪
   - 正当防卫/紧急避险

2. **罪轻辩护**
   - 此罪与彼罪
   - 犯罪情节轻微
   - 自首/立功

3. **量刑辩护**
   - 从轻处罚情节
   - 减轻处罚情节
   - 免除处罚情节

### 步骤六：法律意见书撰写

#### 基本格式
```
法律意见书

致：XXX公司

XXX律师事务所接受贵公司委托，就XXX事项进行法律分析，并出具本法律意见书。

一、事实概述
（概述委托事项的基本事实）

二、法律分析
（进行法律分析，提出法律意见）

三、风险提示
（提示可能存在的法律风险）

四、建议
（提出具体建议）

五、声明
（声明法律意见的局限性）

本法律意见书仅供贵公司参考，不作为法律依据。

XXX律师事务所
律师：XXX
年 月 日
```

#### 撰写要点
1. **事实概述**
   - 客观全面
   - 重点突出
   - 逻辑清晰

2. **法律分析**
   - 法律依据明确
   - 分析透彻
   - 结论明确

3. **风险提示**
   - 风险识别
   - 风险评估
   - 风险防范

4. **建议**
   - 具体可行
   - 有针对性
   - 可操作

## 质量标准

### 文书质量
1. **格式规范** — 符合法院要求
2. **事实清楚** — 陈述客观、逻辑清晰
3. **法律依据明确** — 引用准确、论证有力
4. **语言规范** — 用语准确、表达清晰

### 引用规范
1. **法条引用** — 全称+条号+内容
2. **案例引用** — 案号+法院+裁判要旨
3. **来源标注** — 检索平台+检索时间

## 安全要求

- 当事人敏感信息脱敏处理
- 文书内容保密
- 不对外泄露案件信息
