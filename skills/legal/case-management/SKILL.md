# 案件管理技能

面向中国诉讼律师的案件管理框架，提供从时效管控到案件追踪的全流程支持。

## 适用场景

- 时效/期限管控
- 文书档案管理
- 案件进度追踪
- 案件统计分析

## 执行步骤

### 步骤一：时效/期限管控

#### 时效类型
1. **诉讼时效**
   - 普通诉讼时效：3年
   - 短期诉讼时效：1年
   - 长期诉讼时效：20年

2. **除斥期间**
   - 撤销权期间：1年
   - 解除权期间：1年
   - 追认权期间：1年

3. **上诉期限**
   - 民事上诉期：15天
   - 行政上诉期：15天
   - 刑事上诉期：10天

4. **申请执行期限**
   - 申请执行期：2年
   - 申请恢复执行期：2年

#### 期限管控方法
1. **建立期限台账**
   - 记录所有期限
   - 标注到期时间
   - 设置提醒

2. **设置提醒机制**
   - 提前30天提醒
   - 提前15天提醒
   - 提前7天提醒
   - 提前3天提醒

3. **定期检查**
   - 每周检查
   - 每月检查
   - 每季度检查

### 步骤二：文书档案管理

#### 文书类型
1. **诉讼文书**
   - 起诉状
   - 答辩状
   - 上诉状
   - 代理词

2. **证据材料**
   - 证据清单
   - 证据原件
   - 证据复制件

3. **法院文书**
   - 受理通知书
   - 开庭传票
   - 判决书
   - 裁定书

4. **其他文书**
   - 委托代理合同
   - 授权委托书
   - 律师事务所函

#### 档案管理方法
1. **分类管理**
   - 按案件分类
   - 按文书类型分类
   - 按时间分类

2. **编号管理**
   - 连续编号
   - 便于查找
   - 便于引用

3. **电子化管理**
   - 扫描存档
   - 电子备份
   - 云端存储

### 步骤三：案件进度追踪

#### 进度阶段
1. **立案阶段**
   - 准备起诉材料
   - 提交立案申请
   - 等待受理通知

2. **审理阶段**
   - 等待开庭传票
   - 参加庭审
   - 等待判决

3. **执行阶段**
   - 申请执行
   - 等待执行
   - 执行完毕

#### 进度追踪方法
1. **建立进度台账**
   - 记录每个阶段
   - 标注开始时间
   - 标注结束时间

2. **设置提醒机制**
   - 开庭提醒
   - 举证期限提醒
   - 上诉期限提醒

3. **定期更新**
   - 每周更新
   - 每月更新
   - 每季度更新

### 步骤四：案件统计分析

#### 统计指标
1. **案件数量**
   - 总案件数
   - 进行中案件数
   - 已结案件数

2. **案件类型**
   - 合同纠纷
   - 侵权纠纷
   - 物权纠纷
   - 公司纠纷

3. **案件结果**
   - 胜诉率
   - 败诉率
   - 调解率

4. **案件周期**
   - 平均立案周期
   - 平均审理周期
   - 平均执行周期

#### 统计方法
1. **建立统计台账**
   - 记录每个案件
   - 标注关键指标
   - 定期更新

2. **定期分析**
   - 每月分析
   - 每季度分析
   - 每年分析

3. **改进措施**
   - 分析问题原因
   - 提出改进措施
   - 跟踪改进效果

## 质量标准

### 管理质量
1. **及时性** — 是否及时管理
2. **准确性** — 是否准确管理
3. **完整性** — 是否完整管理
4. **有效性** — 是否有效管理

### 时效管控质量
1. **全面性** — 是否全面管控
2. **及时性** — 是否及时提醒
3. **准确性** — 是否准确计算
4. **有效性** — 是否有效管控

## 安全要求

- 案件信息保密
- 当事人隐私保护
- 档案安全保管
