# 批量法条核实工作流

## 适用场景

当需要对网站、文档、知识卡中的大量法条引用进行批量验证时使用。

## 工作流

### 步骤一：提取法条引用

用正则从文件中提取所有法条引用：

```python
import re
# 匹配 "民法典第XXX条"、"劳动合同法第XX条" 等格式
pattern = r'(?:民法典|刑法|公司法|劳动合同法|行政诉讼法|商标法|专利法|著作权法|保险法|海商法|企业破产法|国家赔偿法|农村土地承包法|反不正当竞争法|旅游法|个人信息保护法|行政处罚法)第(\d+)条'
```

### 步骤二：分批验证

每批15-20条，使用 `mcp_yuandian_search_fatiao` 逐条验证。

**验证策略**：
1. **直接验证**：搜索"法规名 第XXX条"，如返回匹配结果则确认
2. **间接验证**：如直接搜索返回空，搜索引用该条文的其他文件（司法解释、法院指引）
3. **交叉验证**：对比北大法宝 `mcp_pkulaw_get_article` 的结果

### 步骤三：输出核实报告

```markdown
| 序号 | 法条引用 | 期望内容 | 核实结果 | 说明 |
|------|---------|---------|---------|------|
| 1 | 民法典第667条 | 借款合同 | ✅正确 | |
| 2 | 民法典第919条 | 旅游合同 | ❌错误 | 实际是委托合同 |
| 3 | 旅游法第65条 | 解除权 | ⚠️未找到 | 间接验证正确 |
```

### 步骤四：修正错误

发现错误后立即修正，不能跳过。

## 已知错误模式

| 错误类型 | 示例 | 预防方法 |
|---------|------|---------|
| 条文号张冠李戴 | 民法典第919条（委托合同）被误引为旅游合同 | 用MCP验证条文实际内容 |
| 条文号过时 | 农村土地承包法第26条（承包期限实际在第21条） | 搜索最新修正版 |
| 2020年民法典编纂后条文重编 | 旧合同法条文号与民法典不同 | 确认引用的是民法典还是旧法 |

## 验证结果统计（2026-06-09实测）

- 第一轮：37条引用，37/37正确（100%）
- 第二轮：34条引用，28/34正确（82%），2处错误已修正，4处待人工复核
- **总结**：批量生成的法条引用错误率约6%，必须逐条核实
