# 律师庭审实时辅助

## 核心定位

**为律师开庭提供秒级现场辅助**——输入关键词，毫秒级返回法条原文+典型案例+操作建议。

与现有技能的关系：
- trial-preparation（庭前准备）→ 本技能是庭审现场版
- trial-response（庭审应对）→ 本技能是实战执行版
- legal-research（法律检索）→ 本技能是快速调用版
- evidence-organization（证据组织）→ 本技能是质证现场版

## 一、庭审现场快速检索

### 1.1 法条秒查

**触发方式**：输入"查法条"或直接输入法条关键词

**调用工具**：
1. 元典智库 `search_fatiao`（优先，精确到条）
2. 北大法宝 `search_article`（补充，语义搜索）
3. 交叉验证确认时效性

**输出格式**：
```
📋 法条原文
━━━━━━━━━━━━━━━━━━━━━━
《民法典》第584条：
当事人一方不履行合同义务或者履行合同义务不符合约定，造成对方损失的，
损失赔偿额应当相当于因违约所造成的损失，包括合同履行后可以获得的利益；
但是，不得超过违约一方订立合同时预见到或者应当预见到的因违约可能造成的损失。

📌 适用要点
━━━━━━━━━━━━━━━━━━━━━━
- 赔偿范围：实际损失 + 可得利益损失
- 限制：可预见性规则（订立合同时）
- 举证责任：原告需证明损失金额及因果关系

🔗 来源：元典智库 + 北大法宝交叉验证
```

## 二、质证现场辅助

### 2.1 质证要点速查

对方提交证据时，输入"质证：[证据名称]"，即时返回质证要点。

**三性质证模板**：

| 质证维度 | 质证要点 | 话术模板 |
|----------|---------|---------|
| **真实性** | 是否原件/原物？有无篡改？ | "对该证据的真实性不予认可，因为……" |
| **合法性** | 取证方式是否合法？形式是否合规？ | "对该证据的合法性有异议，因为……" |
| **关联性** | 与待证事实有无关联？ | "对该证据的关联性有异议，因为……" |

### 2.2 常见证据质证话术

**合同类证据**：
```
真实性：
- "该合同系复印件，无法核实其真实性"
- "该合同未经双方签章确认"
- "合同签订日期与实际不符"

合法性：
- "该合同签订时存在欺诈/胁迫情形"
- "合同条款违反法律强制性规定"

关联性：
- "该合同与本案争议焦点无关"
- "该合同已被后续协议变更/解除"
```

**转账凭证**：
```
真实性：
- "该转账记录未显示收款方信息"
- "转账金额与主张金额不符"

关联性：
- "该转账系其他业务往来，与本案无关"
- "无法证明该转账与本案合同的关联性"
```

**聊天记录**：
```
真实性：
- "该聊天记录未经公证保全"
- "聊天对象身份无法确认"
- "聊天记录存在删减、篡改可能"

合法性：
- "该聊天记录系非法获取"
- "涉及个人隐私，取证方式不合法"

关联性：
- "聊天内容与本案争议焦点无关"
```

**证人证言**：
```
真实性：
- "证人与对方当事人存在利害关系"
- "证人未亲身感知案件事实"
- "证人证言与其他证据矛盾"

合法性：
- "证人未出庭接受质询"
- "证人证言系猜测、推断"
```

### 2.3 证据异议模板

```
审判长，[原告/被告]对[被告/原告]提交的[证据名称]发表如下质证意见：

1. 关于真实性：[具体意见]
2. 关于合法性：[具体意见]
3. 关于关联性：[具体意见]

综上，[对该证据不予认可/该证据不能证明对方主张]。
```

## 三、对方抗辩即时反驳

### 3.1 常见抗辩反驳速查

| 对方抗辩 | 反驳依据 | 法条支撑 |
|----------|---------|---------|
| 诉讼时效已过 | 时效中断/中止事由 | 民法典第195条 |
| 合同无效 | 合同有效要件 | 民法典第143条 |
| 不可抗力 | 不可抗力认定标准 | 民法典第180条 |
| 混合过错 | 过错比例认定 | 民法典第1173条 |
| 损失无依据 | 损失计算方法 | 民法典第584条 |
| 已部分履行 | 履行情况核实 | 民法典第509条 |

### 3.2 诉讼时效反驳

**对方主张**："原告的诉讼请求已超过诉讼时效"

**反驳话术**：
```
审判长，关于诉讼时效问题，[被告/原告]认为：

1. 诉讼时效期间：根据《民法典》第188条，普通诉讼时效期间为三年。

2. 时效中断事由：根据《民法典》第195条，有下列情形之一的，诉讼时效中断：
   （一）权利人向义务人提出履行请求；
   （二）义务人同意履行义务；
   （三）权利人提起诉讼或者申请仲裁；
   （四）与提起诉讼或者申请仲裁具有同等效力的其他情形。

3. 本案具体情况：[说明时效中断/中止的具体事实]

综上，原告的诉讼请求未超过诉讼时效。
```

### 3.3 合同无效反驳

**对方主张**："该合同违反法律强制性规定，应属无效"

**反驳话术**：
```
审判长，关于合同效力问题：

1. 合同有效要件：根据《民法典》第143条，具备下列条件的民事法律行为有效：
   （一）行为人具有相应的民事行为能力；
   （二）意思表示真实；
   （三）不违反法律、行政法规的强制性规定，不违背公序良俗。

2. 强制性规定的区分：根据《九民纪要》第30条，
   强制性规定包括效力性强制性规定和管理性强制性规定。
   只有违反效力性强制性规定的合同才无效。

3. 本案具体情况：[说明合同为何有效]

综上，该合同合法有效，对双方均有约束力。
```

## 四、赔偿计算速查

### 4.1 劳动纠纷赔偿

**输入**："帮我算赔偿金：月工资X，工作Y年，[违法辞退/经济补偿]"

**计算公式**：
```
经济补偿（N）= 月工资 × 工作年限
  - 每满1年 → 1个月
  - 6个月以上不满1年 → 1个月
  - 不满6个月 → 0.5个月

违法解除赔偿（2N）= 经济补偿 × 2

示例：月工资8000，工作5年
  经济补偿 = 8000 × 5 = 40,000元
  违法解除赔偿 = 40,000 × 2 = 80,000元
```

**法律依据**：
- 《劳动合同法》第47条（经济补偿标准）
- 《劳动合同法》第87条（违法解除双倍赔偿）

### 4.2 民间借贷利息

**输入**："帮我算利息：本金X，年利率Y，借期Z个月"

**计算公式**：
```
利息 = 本金 × 年利率 × 借款月数 / 12

利率上限 = LPR × 4（当前约15.4%）

示例：本金100,000，年利率12%，借期24个月
  利息 = 100,000 × 12% × 24 / 12 = 24,000元
```

**法律依据**：
- 《最高人民法院关于审理民间借贷案件适用法律若干问题的规定》第25条

### 4.3 交通事故赔偿

**输入**："帮我算交通事故赔偿：[伤残等级X，医疗费Y，误工天数Z]"

**计算项目**：
```
1. 医疗费：实际发生
2. 误工费 = 日工资 × 误工天数
3. 护理费 = 护理人员日工资 × 护理天数
4. 伤残赔偿金 = 上年度居民人均可支配收入 × 20年 × 伤残系数
   - 一级100%，二级90%，……十级10%
5. 精神损害抚慰金：根据伤残等级酌定
6. 被扶养人生活费：根据扶养人数和伤残系数计算
```

### 4.4 消费者赔偿

```
假一赔三（最低500元）= 商品价款 × 3
食品假一赔十（最低1000元）= 商品价款 × 10
```

## 五、程序异议依据

### 5.1 管辖异议

**触发**："管辖异议"或"管辖权"

**法律依据**：
```
《民事诉讼法》第22条：对公民提起的民事诉讼，由被告住所地人民法院管辖
《民事诉讼法》第24条：因合同纠纷提起的诉讼，由被告住所地或者合同履行地人民法院管辖
《民事诉讼法》第34条：合同或者其他财产权益纠纷的当事人可以书面协议选择管辖法院
```

**异议话术**：
```
审判长，[被告]对本案管辖权提出异议：

1. 根据《民事诉讼法》第[XX]条，本案应由[XX]法院管辖。
2. [具体理由]
3. 请求将本案移送至[XX]人民法院审理。
```

### 5.2 回避申请

**触发**："回避"或"法官回避"

**法律依据**：
```
《民事诉讼法》第44条：审判人员有下列情形之一的，应当自行回避：
（一）是本案当事人或者当事人、诉讼代理人近亲属的；
（二）与本案有利害关系的；
（三）与本案当事人、诉讼代理人有其他关系，可能影响对案件公正审理的。
```

### 5.3 证据保全

**触发**："证据保全"

**法律依据**：
```
《民事诉讼法》第84条：在证据可能灭失或者以后难以取得的情况下，
当事人可以在诉讼过程中向人民法院申请保全证据。
```

### 5.4 财产保全

**触发**："财产保全"

**法律依据**：
```
《民事诉讼法》第104条：利害关系人因情况紧急，不立即申请保全将会使其合法权益受到难以弥补的损害的，
可以在提起诉讼或者申请仲裁前向被保全财产所在地、被申请人住所地或者对案件有管辖权的人民法院申请采取保全措施。
```

## 七、庭审流程速查

### 7.1 民事庭审流程

```
1. 开庭准备
   - 核对当事人身份
   - 宣布法庭纪律
   - 告知诉讼权利义务

2. 法庭调查
   - 原告陈述诉讼请求及事实理由
   - 被告答辩
   - 举证质证（原告→被告→第三人）
   - 法官发问

3. 法庭辩论
   - 原告发表辩论意见
   - 被告发表辩论意见
   - 第三人发表辩论意见
   - 互相辩论

4. 最后陈述
   - 原告最后陈述
   - 被告最后陈述
   - 第三人最后陈述

5. 调解
   - 法庭询问是否愿意调解
   - 如同意，进行调解

6. 宣判
   - 当庭宣判或择期宣判
```

### 7.2 刑事庭审流程

```
1. 开庭准备
   - 核对被告人身份
   - 宣布法庭纪律
   - 告知诉讼权利义务

2. 法庭调查
   - 公诉人宣读起诉书
   - 被告人陈述
   - 举证质证（公诉人→辩护人）
   - 法官发问

3. 法庭辩论
   - 公诉人发表公诉意见
   - 被告人自行辩护
   - 辩护人发表辩护意见
   - 互相辩论

4. 被告人最后陈述

5. 宣判
```

## 八、庭审注意事项

### 8.1 发言规范

```
称呼：审判长、审判员（合议庭）/ 审判员（独任庭）
开头：[原告/被告/辩护人]发表如下意见……
结尾：以上意见，恳请合议庭/审判员予以采纳。
```

### 8.2 禁止行为

- ❌ 不得对法官使用不敬语言
- ❌ 不得打断对方发言（需举手示意）
- ❌ 不得向证人诱导性发问
- ❌ 不得泄露国家秘密、商业秘密、个人隐私
- ❌ 不得进行人身攻击

### 8.3 举证时限

```
《民事诉讼法》第68条：当事人对自己提出的主张应当及时提供证据。
人民法院根据当事人的主张和案件审理情况，确定当事人应当提供的证据及其期限。

举证期限届满后，当事人对已经提供的证据，申请提供反驳证据或者对证据来源、
形式等方面的瑕疵进行补正的，人民法院可以酌情再次确定举证期限。
```
