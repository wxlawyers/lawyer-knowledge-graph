# 劳动补偿知识技能

## 一、经济补偿

### 1.1 计算公式

```
经济补偿（N）= 月工资 × 工作年限

月工资：劳动者在劳动合同解除或者终止前12个月的平均工资
工作年限：
- 每满1年 → 1个月
- 6个月以上不满1年 → 1个月
- 不满6个月 → 0.5个月
```

### 1.2 适用情形

| 情形 | 补偿标准 | 法律依据 |
|------|---------|---------|
| 协商解除（用人单位提出） | N | 第36条 |
| 劳动者被迫辞职 | N | 第38条 |
| 无过失性辞退 | N+1（代通知金） | 第40条 |
| 经济性裁员 | N | 第41条 |
| 合同期满不续签 | N | 第44条 |
| 违法解除 | 2N | 第87条 |

### 1.3 月工资上限

- 月工资高于当地上年度职工月平均工资3倍的，按3倍计算
- 补偿年限最高不超过12年

## 二、赔偿金

### 2.1 违法解除赔偿

```
赔偿金 = 经济补偿 × 2 = 月工资 × 工作年限 × 2
```

**适用情形**：
- 用人单位违法解除劳动合同
- 用人单位违法终止劳动合同

### 2.2 代通知金

```
代通知金 = 1个月工资
```

**适用情形**：
- 用人单位未提前30日书面通知解除劳动合同

## 三、社保公积金

### 3.1 社保缴费基数

```
上限：当地上年度职工月平均工资 × 3
下限：当地上年度职工月平均工资 × 60%
```

### 3.2 社保缴费比例

| 险种 | 单位比例 | 个人比例 |
|------|---------|---------|
| 养老保险 | 16% | 8% |
| 医疗保险 | 8% | 2% |
| 失业保险 | 0.5% | 0.5% |
| 工伤保险 | 0.2%-1.9% | 0 |
| 生育保险 | 0.5% | 0 |

### 3.3 公积金

- 缴存比例：5%-12%
- 缴存基数：上年度月平均工资
- 上限：当地上年度职工月平均工资 × 3

### 3.4 社保补缴

**法律依据**：
- 《社会保险法》第63条：用人单位未按时足额缴纳社会保险费的，由社会保险费征收机构责令其限期缴纳或者补足

**补缴流程**：
1. 向社保经办机构投诉
2. 社保经办机构责令用人单位补缴
3. 用人单位逾期不缴纳的，加收滞纳金

## 四、工伤赔偿

### 4.1 工伤认定

**认定条件**：
- 在工作时间和工作场所内，因工作原因受到事故伤害
- 工作时间前后在工作场所内，从事与工作有关的预备性或者收尾性工作受到事故伤害
- 在工作时间和工作场所内，因履行工作职责受到暴力等意外伤害
- 患职业病的
- 因工外出期间，由于工作原因受到伤害或者发生事故下落不明
- 在上下班途中，受到非本人主要责任的交通事故或者城市轨道交通、客运轮渡、火车事故伤害

### 4.2 伤残等级赔偿

| 等级 | 一次性伤残补助金 | 伤残津贴（月） |
|------|----------------|--------------|
| 一级 | 27个月工资 | 90%工资 |
| 二级 | 25个月工资 | 85%工资 |
| 三级 | 23个月工资 | 80%工资 |
| 四级 | 21个月工资 | 75%工资 |
| 五级 | 18个月工资 | 70%工资 |
| 六级 | 16个月工资 | 60%工资 |
| 七级 | 13个月工资 | 无 |
| 八级 | 11个月工资 | 无 |
| 九级 | 9个月工资 | 无 |
| 十级 | 7个月工资 | 无 |

### 4.3 工亡赔偿

```
丧葬补助金：6个月统筹地区上年度职工月平均工资
供养亲属抚恤金：
- 配偶：40%工资
- 其他亲属：30%工资
- 孤儿/孤寡老人：增加10%
一次性工亡补助金：上一年度全国城镇居民人均可支配收入 × 20倍
```

**2025年标准**：
- 一次性工亡补助金：约100万元

## 五、加班费

### 5.1 计算标准

| 类型 | 倍数 | 法律依据 |
|------|------|---------|
| 工作日延长 | 1.5倍 | 第44条第1项 |
| 休息日加班 | 2倍 | 第44条第2项 |
| 法定节假日 | 3倍 | 第44条第3项 |

### 5.2 计算公式

```
小时工资 = 月工资 ÷ 21.75天 ÷ 8小时

工作日加班费 = 小时工资 × 加班小时数 × 1.5
休息日加班费 = 小时工资 × 加班小时数 × 2
法定节假日加班费 = 小时工资 × 加班小时数 × 3
```

### 5.3 举证责任

- 用人单位掌握加班事实存在的证据，用人单位不提供的，由用人单位承担不利后果

## 六、年假工资

### 6.1 年假天数

| 工龄 | 年假天数 |
|------|---------|
| 1-10年 | 5天 |
| 10-20年 | 10天 |
| 20年以上 | 15天 |

### 6.2 未休年假工资

```
未休年假工资 = 日工资 × 未休天数 × 200%

日工资 = 月工资 ÷ 21.75天
```

## 七、速查表

### 7.1 经济补偿速查

```
协商解除：N
无过失辞退：N+1
违法解除：2N

示例：月工资8000，工作5年
- 协商解除：8000 × 5 = 40,000元
- 无过失辞退：8000 × 5 + 8000 = 48,000元
- 违法解除：8000 × 5 × 2 = 80,000元
```

### 7.2 工伤赔偿速查

```
十级伤残：7个月工资 + 医疗费 + 停工留薪期工资
九级伤残：9个月工资 + 医疗费 + 停工留薪期工资
八级伤残：11个月工资 + 医疗费 + 停工留薪期工资
```

### 7.3 加班费速查

```
工作日加班：1.5倍
休息日加班：2倍
法定节假日：3倍

示例：月工资8000，工作日加班10小时
小时工资 = 8000 ÷ 21.75 ÷ 8 = 45.98元
加班费 = 45.98 × 10 × 1.5 = 689.7元
```
