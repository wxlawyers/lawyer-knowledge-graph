# 降AI味检查清单

## AI典型用语（必须替换）

| AI用语 | 替换为 |
|--------|--------|
| 值得注意的是 | 删除，直接说 |
| 综上所述 | 总结一下 / 说白了 |
| 总而言之 | 一句话说 |
| 需要强调的是 | 删除，用加粗代替 |
| 不难发现 | 你看 / 明眼人一看就懂 |
| 由此可见 | 所以 / 说到底 |
| 在实践中 | 我办案中 / 实际操作中 |
| 根据相关规定 | 根据《XXX》第X条 |
| 一般来说 | 通常 / 大多数情况下 |
| 从法律角度来看 | 法律上讲 |
| 建议当事人 | 你该怎么做 |
| 具体而言 | 具体来说 / 举个例子 |
| 换言之 | 换句话说 / 说白了 |
| 众所周知 | 删除，直接说 |
| 综合考虑 | 想来想去 / 权衡利弊 |

## 降AI味技巧

### 1. 加入个人执业经历
- ❌ "在实践中，合同纠纷案件通常涉及..."
- ✅ "我执业15年，接过上百个合同纠纷案子，最常见的就是..."

### 2. 用口语化表达
- ❌ "根据《中华人民共和国公司法》的相关规定，股东应当..."
- ✅ "公司法说得很清楚，股东必须..."

### 3. 加入个人观点
- ❌ "虚假出资可能面临刑事处罚。"
- ✅ "说实话，很多老板根本不知道，注册资本填太高真的会坐牢。"

### 4. 用案例引入
- ❌ "本文将探讨注册资本实缴问题。"
- ✅ "去年我接了一个案子，海门一个老板来找我，一进门就说：'余律师，我可能要坐牢了。'"

### 5. 短句化
- ❌ "鉴于新公司法的实施以及相关配套法规的出台，企业应当及时调整其注册资本策略以确保合规经营。"
- ✅ "新公司法已经实施了。你的注册资本策略该调整了。"

### 6. 避免过度结构化
- ❌ 每段都是"第一...第二...第三..."
- ✅ 穿插案例、反问、感叹，让文章有节奏感

## 校对重点

1. **法条条文号**：必须与原文一致
2. **案号格式**：（年份）法院简称案件类型字第XXX号
3. **金额数字**：大写小写一致
4. **日期**：年月日格式统一
5. **当事人信息**：确认已脱敏
6. **引用出处**：确认已标注
