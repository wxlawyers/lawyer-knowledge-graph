### 步骤二：法律科普内容输出

#### Hermes指令速查

**公众号文章**：
```
帮我写一篇关于[主题]的公众号文章，要求：
1. 标题20字以内，制造好奇
2. 用案例开头，3句话代入
3. 正文结构：痛点→法律依据→案例→怎么做
4. 字数400-800字
5. 口语化，不卖弄法条
6. 结尾引导关注
```

**短视频脚本**：
```
帮我写一个关于[主题]的短视频脚本，要求：
1. 时长30-60秒
2. 开头3秒抓注意力
3. 用大白话讲法律知识
4. 包含真实或虚构案例
5. 结尾引导关注
6. 输出分镜表格式
```

**热点评论**（结合每日法律速报）：
```
基于今天的法律热点，帮我写一条[平台]的法律评论：
- 200字以内
- 专业但不卖弄
- 结尾引导关注
```

### 步骤五：直播运营

律师直播是当前最有效的案源获取渠道之一。完整的直播运营体系包括：平台规则合规、内容策划、话术模板、SOP流程、数据复盘、引流转化。

#### 直播平台选择

| 平台 | 优势 | 适合阶段 |
|------|------|----------|
| 抖音 | 流量大、推荐机制强 | 主力平台 |
| 视频号 | 微信生态、私域转化好 | 辅助平台 |
| 快手 | 下沉市场、粘性高 | 可选 |

#### 律师直播核心合规红线

1. **绝对禁止**：承诺案件结果、泄露当事人隐私、诋毁同行、虚假法律信息
2. **必备声明**：直播间简介+每30分钟口头声明"本内容仅供参考，不构成法律意见"
3. **引流合规**：不能在直播间直接说微信号，使用平台官方"咨询"组件

#### 直播内容矩阵

| 内容类型 | 占比 | 目的 |
|----------|------|------|
| 案例故事 | 40% | 吸引关注、建立信任 |
| 知识科普 | 30% | 价值输出、涨粉 |
| 热点解读 | 15% | 蹭热度、破圈 |
| 互动答疑 | 15% | 增强粘性、转化 |

#### 直播转化漏斗

```
公域流量 → 直播间观看 → 关注账号 → 私信互动 → 添加微信 → 线下咨询 → 委托案件
```

### 步骤六：命理赋能客户关系

懂命理的律师在企业家客户圈里有独特优势——企业家群体普遍信命理风水，你懂这个，信任感直接拉满。

#### 核心定位

命理是**破冰工具**，不是业务核心。用命理建立信任，用专业解决问题。

#### 业务应用场景

| 场景 | 操作 | 效果 |
|------|------|------|
| 企业家破冰 | 聊八字拉近距离 | 信任感+100% |
| 合伙纠纷 | 分析八字相冲 | 客户理解冲突根源 |
| 签约择日 | 选黄道吉日 | 客户觉得专业 |
| 短视频选题 | 法律+命理结合 | 差异化内容 |

#### 学习路径（3周入门）

1. **入门（1-2周）**：天干地支、五行生克、会用APP排盘
2. **进阶（2-4周）**：十神体系、格局判断、大运流年
3. **实战（持续）**：客户性格分析、沟通策略调整

#### 十神看客户类型速查

| 十神 | 客户特征 | 沟通策略 |
|------|----------|----------|
| 正官型 | 守规矩、重名声 | 正式、专业、给面子 |
| 七杀型 | 有魄力、敢冒险 | 直接、果断、给方案 |
| 正财型 | 务实、重利益 | 讲费用、给数字 |
| 偏财型 | 大方、好面子 | 给面子、捧着聊 |
| 伤官型 | 叛逆、有主见 | 平等、尊重、不说教 |
| 劫财型 | 争强好胜 | 支持立场、强调维权 |

#### 短视频选题（法律+命理）

- "做生意的人，这3种八字最容易惹官司"
- "合伙创业，先看看八字合不合"
- "签合同选日子，是迷信还是有道理？"
