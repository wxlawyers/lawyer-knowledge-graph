# 知识沉淀与复用技能

面向中国诉讼律师的知识管理框架，提供从案件归档到知识复用的全流程支持。

## 适用场景

- 案件归档
- 裁判规则积累
- 文书模板库维护
- 知识检索复用

## 执行步骤

### 步骤一：案件归档

#### 归档原则
1. **完整性**
   - 归档所有案件材料
   - 不遗漏重要内容
   - 保持材料完整

2. **系统性**
   - 分类归档
   - 编号管理
   - 便于查找

3. **安全性**
   - 妥善保管
   - 防止丢失
   - 防止泄露

#### 归档内容
1. **案件基本信息**
   - 案号
   - 案由
   - 当事人
   - 管辖法院

2. **诉讼文书**
   - 起诉状
   - 答辩状
   - 代理词
   - 判决书

3. **证据材料**
   - 证据清单
   - 证据原件
   - 证据复制件

4. **办案记录**
   - 会见笔录
   - 调查笔录
   - 庭审笔录

5. **办案经验**
   - 案件分析
   - 办案策略
   - 经验教训

#### 归档方法
1. **电子化归档**
   - 扫描存档
   - 电子备份
   - 云端存储

2. **分类归档**
   - 按案件类型分类
   - 按时间分类
   - 按当事人分类

3. **编号归档**
   - 连续编号
   - 便于查找
   - 便于引用

## 质量标准

### 归档质量
1. **完整性** — 是否完整归档
2. **系统性** — 是否系统归档
3. **安全性** — 是否安全保管
4. **可检索性** — 是否便于检索

### 知识积累质量
1. **全面性** — 是否全面积累
2. **准确性** — 是否准确积累
3. **时效性** — 是否及时更新
4. **有效性** — 是否有效积累

## 安全要求

- 案件信息保密
- 当事人隐私保护
- 知识产权保护
