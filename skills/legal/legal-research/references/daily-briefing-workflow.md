# 每日法律速报 — 详细执行流程

## 执行时序

### 阶段一：并行采集（delegate_task，约70秒）

使用 `delegate_task` 启动 3 个并行子代理：

**子代理 1 — 江苏法院网**
- 目标：https://www.jsfy.gov.cn
- 采集：重要新闻、案例速递、司法文件、各地动态
- 返回：结构化摘要

**子代理 2 — 中国法院网 + 司法部**
- 目标：https://www.chinacourt.org + https://www.moj.gov.cn
- 采集：最新司法解释、典型案例、立法动态、规章发布
- 返回：结构化摘要

**子代理 3 — MCP 工具检索**
- 工具：`mcp_yuandian_search_fagui`（法规）、`mcp_yuandian_search_qwal`（案例）
- 关键词：按月份（如"2026年6月"）、按领域（如"商事调解"）
- 返回：结构化检索结果

### 阶段二：权威来源补充（主代理，约60秒）

主代理直接浏览器采集：

**最高人民法院 court.gov.cn**
- 导航：首页 → 权威发布
- 重点栏目：司法解释（含日期）、重大案件、司法文件、司法数据
- 采集方法：阅读权威发布页面的列表，提取近期条目

**最高人民检察院 spp.gov.cn**
- 导航：首页 → 检察新闻 / 权威发布
- 重点：典型案例、检察政策、公诉动态

### 阶段三：MCP 交叉验证

**元典智库验证**
```
mcp_yuandian_search_fagui(keyword="法规关键词")
mcp_yuandian_search_fatiao(keyword="法规名称 第XXX条")
```

**北大法宝验证**（如服务可用）
```
mcp_pkulaw_search_article(text="关键词")
mcp_pkulaw_get_article(title="法规名称", number="第XXX条")
```

### 阶段四：Obsidian 输出

**路径规范**
```
~/Library/Mobile Documents/iCloud~md~obsidian/Documents/法律知识库/09-每日法律速报/{YYYY-MM-DD}.md
```

**内容结构**（参考 `templates/daily-legal-report.md`）
1. 🔥 今日热点（按重要性排序，含事件概述、来源、律师执业关联性）
2. ⚖️ 最新案例（案由、裁判要旨、实务启示）
3. 📢 法规动态（表格：法规名称、文号、生效日期、摘要）
4. 📢 江苏法院动态（本地司法动态优先）
5. 💡 实务提示（对律师执业的具体建议）
6. 📦 知识沉淀清单（标记待沉淀到各知识库模块）

**⚠️ 标注规则**
- 新法规即将生效（7天内）→ 速报开头用 ⚠️ 标注
- 最高法/最高检发布重大典型案例 → 速报开头用 ⚠️ 标注
- 江苏法院系统重大政策变化 → 速报开头用 ⚠️ 标注

### 阶段五：Excalidraw 配图

**文件命名**
```
{YYYY-MM-DD}-热点导图.excalidraw
{YYYY-MM-DD}-时间线.excalidraw
{YYYY-MM-DD}-案例图.excalidraw
```

**Excalidraw 要点**
- 使用 container binding 方式添加文字（boundElements + containerId）
- 不要使用 label 属性（会被忽略）
- roughness: 1（手绘风格）
- 文字最小 16px，标题 20-24px

## MCP 工具在每日速报中的已知限制

### 元典智库 search_qwal（案例检索）
- **问题**：对基于日期的查询（如"2026年6月 典型案例"）返回完全不相关的旧案例
- **实测**：搜索"2026年6月 典型案例"返回的是2007-2010年的专利、著作权案例
- **结论**：`search_qwal` 不能用于"最新案例"检索，仅适合按具体案由+关键词检索
- **替代方案**：浏览器直接采集最高法/最高检官网的案例列表

### 元典智库 search_fagui（法规检索）
- **可靠性**：对主流法规、司法解释检索效果较好
- **日期检索**：按月份搜索可返回相关结果，但可能包含非法规类文件（如培训通知、汇率公告）
- **建议**：结合具体法规名称搜索，而非仅按日期

### 北大法宝 MCP 服务
- **稳定性**：HTTP 服务可能返回 "MCP server not connected" 错误
- **降级策略**：如不可用，跳过交叉验证步骤，在报告中注明

## Excalidraw 配图模式

### 模式一：热点思维导图
- 中心节点：日期 + "法律热点"
- 分支：各热点事件（颜色区分类型）
- 子分支：关键信息（如条文数量、施行日期）
- 颜色：热点=蓝色(#a5d8ff)、案例=绿色(#b2f2bb)、法规=橙色(#ffd8a8)、江苏=紫色(#c3fae8)

### 模式二：法规生效时间线
- 水平时间轴（矩形线段 + 圆形节点）
- 节点上方：日期和法规名称（圆角矩形，颜色按类型区分）
- 底部：图例说明
- 按时间顺序从左到右排列

### 模式三：案例关系图
- 按类型分区（刑事、公益诉讼、司法解释、本地案例）
- 每区一个标题栏（带颜色）
- 每个案例一个圆角矩形卡片
- 卡片内：案由、核心信息、实务启示

## 执行经验

### 并行采集效率
- 3 个子代理并行执行，总耗时约 70 秒（最长子代理耗时）
- 比串行执行快 2-3 倍

### 主浏览器采集
- 最高法权威发布页面结构清晰，日期格式标准
- 最高检首页滚动新闻需要仔细阅读
- 江苏法院网更新频率可能不覆盖周末

### 输出文件大小
- 主 Markdown 文件：约 8-12KB
- 每个 Excalidraw 文件：约 8-12KB
- 总输出：约 35-50KB（4个文件）
