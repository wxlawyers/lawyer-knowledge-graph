# 法律检索策略

## 检索原则

### 三轮递进
1. **第一轮**：元典智库 + 北大法宝双源交叉检索
2. **第二轮**：人民法院案例库验证裁判规则
3. **第三轮**：威科先行/裁判文书网补充类案

### 双源交叉验证
- 法条引用必须双源验证
- 案例检索多平台对比
- 标注来源和检索时间

## 检索工具

### 元典智库（5个工具）

#### 法规检索
```
mcp_yuandian_search_fagui(keyword="民法典")
```
- 用途：检索法律法规
- 返回：法规标题、发布部门、施行日期、时效性

#### 法条检索
```
mcp_yuandian_search_fatiao(keyword="民法典 第五百七十七条")
```
- 用途：精确检索法条内容
- 建议：使用"法规名称 条号"格式

#### 案例检索
```
mcp_yuandian_search_qwal(keyword="合同纠纷 违约 南京 2024")
```
- 用途：检索司法案例
- 建议：包含案由+争议焦点+地域+年份

#### 法规详情
```
mcp_yuandian_get_fagui_detail(id="法规ID")
```
- 用途：获取法规全文

#### 法条详情
```
mcp_yuandian_get_fatiao_detail(id="法条ID")
```
- 用途：获取法条完整内容

### 北大法宝（9个服务）

#### 法规搜索

mcp_pkulaw_search_law(keyword="民法典")
```

#### 案例检索

```

## 检索策略

### 策略一：请求权基础检索

**目的**：确定请求权基础的法律依据

**步骤**：
1. 确定法律关系性质（合同/侵权/物权/公司）
2. 检索相关法规（民法典合同编/侵权责任编/物权编/公司法）
3. 检索具体法条（违约责任/侵权责任/物权保护/股东权利）
4. 交叉验证法条有效性

**示例**：
```
# 合同纠纷
mcp_yuandian_search_fagui(keyword="民法典 合同")
mcp_yuandian_search_fatiao(keyword="民法典 违约责任")
mcp_pkulaw_search_law(keyword="民法典 合同编")
```

### 策略二：类案检索

**目的**：查找支持我方主张的类案

**步骤**：
1. 确定案由和争议焦点
2. 组合关键词检索（案由+争议焦点+地域+年份）
3. 提取裁判要旨
4. 验证案例有效性

**示例**：
```
# 股权转让纠纷
mcp_yuandian_search_qwal(keyword="股权转让 股东优先购买权 南京 2024")
mcp_pkulaw_search_case(keyword="股权转让 优先购买权")
```

### 策略三：裁判规则验证

**目的**：验证裁判规则的适用性

**步骤**：
1. 检索相关案例
2. 提取裁判要旨
3. 对比我方案情
4. 判断是否适用

**示例**：
```
# 检索案例
mcp_yuandian_search_qwal(keyword="合同解除 违约")
# 提取裁判要旨
# 对比我方案情
# 判断是否适用
```

## 检索关键词组合

### 案由 + 争议焦点
- 合同纠纷 + 违约责任
- 侵权纠纷 + 因果关系
- 物权纠纷 + 物权变动
- 公司纠纷 + 股东权利

### 案由 + 地域 + 年份
- 合同纠纷 + 南京 + 2024
- 侵权纠纷 + 江苏 + 2023
- 物权纠纷 + 无锡 + 2024

### 争议焦点 + 法律依据
- 违约责任 + 民法典第五百七十七条
- 侵权责任 + 民法典第一千一百六十五条
- 物权保护 + 民法典第二百三十五条

## 检索结果处理

### 法规检索结果
1. 确认法规时效性（是否现行有效）
2. 确认条文完整性（是否被修改）
3. 标注来源和检索时间

### 案例检索结果
1. 确认案例有效性（是否被撤销）
2. 提取裁判要旨
3. 对比我方案情
4. 判断参考价值

### 法条检索结果
1. 确认法条内容完整性
2. 确认是否被司法解释补充
3. 标注来源和检索时间

## 注意事项

1. **时效性**：确认法规是否现行有效
2. **完整性**：确认法条是否被修改或补充
3. **有效性**：确认案例是否被撤销或改判
4. **关联性**：确认检索结果与本案是否相关
5. **来源标注**：标注检索平台和检索时间
