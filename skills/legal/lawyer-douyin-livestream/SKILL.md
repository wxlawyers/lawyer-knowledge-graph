# 律师抖音直播助手

> 参考资料：`references/douyin-platform-rules.md` — 抖音平台规则、违禁词详细分类、安全替代表达、执业纪律条款、直播事故案例、合规检查清单

## 核心定位

为律师抖音直播提供**全流程辅助**：从选题策划、脚本撰写、直播中实时辅助到直播后复盘。重点解决三个痛点：

1. **话术合规**——违禁词实时检测，避免封号/限流
2. **案例秒查**——输入关键词，毫秒级返回法条+案例+企业信息
3. **转化高效**——从观众到咨询到委托的完整链路设计

## 一、直播前准备

### 1.1 选题策划

**爆款选题公式**：痛点 × 时效 × 争议性

| 选题类型 | 示例 | 流量潜力 |
|----------|------|---------|
| 热点事件法律解读 | "某明星离婚案的法律看点" | ⭐⭐⭐⭐⭐ |
| 高频纠纷科普 | "借钱不还怎么办" | ⭐⭐⭐⭐ |
| 反常识法律知识 | "签了合同也能反悔的5种情况" | ⭐⭐⭐⭐ |
| 职场法律 | "被公司辞退能拿多少赔偿" | ⭐⭐⭐⭐ |
| 婚姻家事 | "离婚时房子怎么分" | ⭐⭐⭐⭐⭐ |
| 刑事风险 | "这些行为你可能不知道是犯罪" | ⭐⭐⭐⭐ |
| 消费维权 | "买到假货怎么索赔" | ⭐⭐⭐ |

**选题来源**：
- 微博热搜法律相关话题
- 中国裁判文书网最新案例
- 北大法宝/元典智库热点法规
- 评论区高频提问汇总
- 同行爆款视频分析

### 1.2 脚本框架

**标准30分钟直播脚本结构**：

```
[0-3分钟] 开场 hook
  - 招呼+今日主题预告
  - 抛出一个悬念/争议问题
  - "今天聊完你可能会改变看法"

[3-8分钟] 痛点共鸣
  - 描述观众可能遇到的场景
  - "是不是有人遇到过这种情况……"
  - 引发弹幕互动

[8-20分钟] 核心内容
  - 法律依据（法条原文+白话翻译）
  - 真实案例（脱敏处理）
  - 操作步骤（"你该怎么做，第一步……"）
  - 常见误区纠正

[20-25分钟] 互动答疑
  - 回答弹幕提问
  - 实时案例检索演示
  - "有人问XX问题，我们来看看法律怎么说"

[25-28分钟] 案源转化
  - "如果你的情况比较复杂，可以……"
  - 引导私信/关注/加粉丝群
  - 预告下期内容

[28-30分钟] 收尾
  - 总结今日要点
  - 感谢+关注引导
  - "下期我们聊XX，记得来"
```

### 1.3 合规自查清单

直播前逐项确认：

- [ ] 脚本无违禁词（用违禁词检测功能扫描）
- [ ] 案例已脱敏（无真实姓名、身份证号、具体地址）
- [ ] 未承诺胜诉结果
- [ ] 未贬低同行/法院/对方当事人
- [ ] 引用法条准确有效（用MCP工具验证时效性）
- [ ] 未涉及国家秘密、商业秘密、个人隐私
- [ ] 已标注"仅供参考，不构成法律意见"

## 二、违禁词体系

### 2.1 绝对禁止（触发封号/限流）

**平台类违禁词**（抖音规则）：
```
保证赢 / 包赢 / 一定能赢 / 必胜
100%能要回来 / 保证能离
包在我身上 / 保证帮你搞定
最厉害 / 最专业 / 第一律师 / 全国第一
没有打不赢的官司
```

**法律类违禁词**（执业规范）：
```
暗示与法官有关系
承诺案件结果
贬低对方当事人/律师
泄露当事人隐私
以不正当手段招揽业务
```

### 2.2 高风险词（需替换表达）

| 高风险词 | 安全替代 |
|----------|---------|
| 保证赢 | 根据法律规定，胜诉的可能性较大 |
| 一定能要回来 | 法律支持你的主张，可以依法追偿 |
| 包离 | 符合法定条件的，法院会判决离婚 |
| 我帮你搞定 | 我可以为你提供法律分析和建议 |
| 最好的办法 | 比较稳妥的处理方式是…… |
| 对方必输 | 对方的法律依据存在明显漏洞 |
| 你放心 | 法律规定对你有利 |
| 肯定没问题 | 从法律角度分析，风险较低 |

### 2.3 直播间敏感话题

| 话题 | 处理方式 |
|------|---------|
| 政治敏感事件 | 不评论，引导到法律分析层面 |
| 正在审理的案件 | 只讨论已公开信息，不预测判决 |
| 涉及未成年人 | 严格脱敏，不透露任何身份信息 |
| 涉及自杀/自残 | 立即引导拨打心理援助热线 |
| 涉及群体事件 | 不煽动情绪，引导理性维权 |
| 医疗纠纷细节 | 不评价医疗行为，引导专业鉴定 |

### 2.4 违禁词检测指令

当用户提供直播话术时，执行以下检测：

```
用户输入 → 扫描以下维度：
1. 平台违禁词（保证/包/一定/最/第一）
2. 执业违规（承诺结果/贬低同行/暗示关系）
3. 敏感话题（政治/正在审理/未成年人）
4. 绝对化表述（所有/任何/绝对/永远）
→ 发现问题 → 标红 + 提供安全替代表达
```

### 5.2 常用法律速查表

**劳动纠纷**：
- 经济补偿：N（每满1年1个月工资）
- 违法解除赔偿：2N
- 未签合同双倍工资：最多11个月
- 仲裁时效：1年（从知道或应当知道之日起）
- 仲裁费用：免费

**婚姻家事**：
- 离婚冷静期：30天
- 共同财产：婚后所得原则上共同所有
- 抚养权：2周岁以下归母亲，2-8周岁综合判断，8周岁以上听孩子意见
- 彩礼返还：未登记/未共同生活/导致生活困难

**民间借贷**：
- 利率上限：LPR的4倍（当前约15.4%）
- 诉讼时效：3年
- 举证责任：原告需证明借贷合意+款项交付
- 借条vs欠条：借条证明力更强

**交通事故**：
- 交强险限额：死亡伤残18万+医疗1.8万+财产0.2万
- 责任划分：全责/主责/同责/次责/无责
- 赔偿项目：医疗费+误工费+护理费+伤残赔偿金+精神损害抚慰金
- 诉讼时效：3年

**消费者维权**：
- 假一赔三（最低500元）
- 食品假一赔十（最低1000元）
- 七天无理由退货
- 举证责任倒置（6个月内）

## 七、注意事项

### 7.1 执业纪律红线

1. **不得承诺案件结果**——只能说"法律规定的可能性"
2. **不得贬低同行**——不评价其他律师的专业能力
3. **不得暗示与司法人员关系**——不提"我认识XX法官"
4. **保守当事人秘密**——直播中不透露任何当事人信息
5. **不得以不正当方式招揽业务**——不夸大宣传、不诋毁同行

### 7.2 平台运营建议

- 固定直播时间，培养观众习惯
- 每期聚焦一个主题，不贪多
- 互动率比在线人数更重要
- 直播后及时剪辑短视频二次传播
- 建立粉丝群，沉淀私域流量

### 7.3 数据来源标注

使用MCP工具检索的结果，直播中需标注：
- "这个案例来自中国裁判文书网"
- "根据北大法宝的检索结果"
- "数据来源于同花顺旗下快查企业数据引擎"
