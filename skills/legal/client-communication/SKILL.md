# 当事人沟通技能

面向中国诉讼律师的当事人沟通框架，提供从案情表达到预期管理的全流程支持。

## 适用场景

- 案情通俗化表达
- 风险坦诚告知
- 预期管理
- 沟通技巧

## 执行步骤

### 步骤一：案情通俗化表达

#### 表达原则
1. **简洁明了**
   - 语言简洁
   - 表达清晰
   - 重点突出

2. **通俗易懂**
   - 避免法律术语
   - 使用日常用语
   - 举例说明

3. **客观真实**
   - 客观陈述
   - 不夸大
   - 不缩小

#### 表达技巧
1. **类比说明**
   - 用生活案例类比
   - 用常见事物类比
   - 用简单逻辑类比

2. **图表说明**
   - 时间线图表
   - 流程图
   - 关系图

3. **案例说明**
   - 类似案例
   - 成功案例
   - 失败案例

#### 案情表达模板
```
案情说明

一、案件基本情况
（用通俗语言描述案件事实）

二、法律关系
（用通俗语言解释法律关系）

三、争议焦点
（用通俗语言说明争议焦点）

四、诉讼请求
（用通俗语言说明诉讼请求）

五、法律依据
（用通俗语言说明法律依据）
```

### 步骤二：风险坦诚告知

#### 告知原则
1. **全面告知**
   - 告知所有风险
   - 不隐瞒风险
   - 不淡化风险

2. **客观告知**
   - 客观分析风险
   - 不夸大风险
   - 不缩小风险

3. **及时告知**
   - 及时告知风险
   - 不拖延告知
   - 不事后告知

#### 风险类型
1. **诉讼风险**
   - 败诉风险
   - 部分败诉风险
   - 执行风险

2. **程序风险**
   - 时效风险
   - 管辖风险
   - 证据风险

3. **成本风险**
   - 诉讼费风险
   - 律师费风险
   - 时间成本风险

```
风险告知书

2. 部分败诉风险：XXX
3. 执行风险：XXX

二、程序风险
1. 时效风险：XXX
2. 管辖风险：XXX
3. 证据风险：XXX

三、成本风险
1. 诉讼费风险：XXX
2. 律师费风险：XXX
3. 时间成本风险：XXX

四、风险防范措施
1. XXX
2. XXX
3. XXX
```

### 步骤三：预期管理

#### 管理原则
1. **现实预期**
   - 建立现实预期
   - 不建立过高预期
   - 不建立过低预期

2. **动态调整**
   - 根据案件进展调整
   - 根据证据变化调整
   - 根据法律变化调整

3. **及时沟通**
   - 及时沟通预期变化
   - 不拖延沟通
   - 不隐瞒变化

#### 预期管理技巧
1. **预期设定**
   - 设定合理预期
   - 设定最低预期
   - 设定最高预期

2. **预期调整**
   - 根据案件进展调整
   - 根据证据变化调整
   - 根据法律变化调整

3. **预期沟通**
   - 及时沟通预期变化
   - 解释变化原因
   - 提供应对方案

#### 预期管理模板

```
三、预期调整
1. 调整原因：XXX
2. 调整内容：XXX
3. 调整后预期：XXX

四、应对方案
1. 最好结果应对：XXX

3. 最差结果应对：XXX
```

### 步骤四：沟通技巧

#### 倾听技巧
1. **认真倾听**
   - 全神贯注
   - 不打断对方
   - 不急于下结论

2. **理解意图**
   - 理解对方真实意图
   - 理解对方情感需求
   - 理解对方利益诉求

3. **反馈确认**
   - 反馈理解内容
   - 确认理解是否正确
   - 避免误解

#### 表达技巧
1. **清晰表达**
   - 语言简洁
   - 表达清晰
   - 重点突出

2. **共情表达**
   - 理解对方感受
   - 表达理解
   - 建立信任

3. **专业表达**
   - 专业分析
   - 专业建议
   - 专业方案

#### 沟通话术模板

```
3. 反馈：XXX

三、表达话术
1. 案情说明：XXX

3. 预期管理：XXX

```

### 步骤五：沟通记录

#### 记录原则
1. **及时记录**
   - 及时记录沟通内容
   - 不拖延记录
   - 不遗漏重要内容

2. **准确记录**
   - 准确记录沟通内容
   - 不歪曲内容
   - 不添加内容

3. **完整记录**
   - 完整记录沟通内容
   - 不遗漏重要内容
   - 不省略重要内容

#### 记录内容
1. **沟通时间**
   - 沟通开始时间
   - 沟通结束时间
   - 沟通时长

2. **沟通方式**
   - 面谈
   - 电话
   - 微信
   - 邮件

3. **沟通内容**
   - 案件情况
   - 风险告知
   - 预期管理
   - 其他内容

4. **沟通结果**
   - 达成共识
   - 存在分歧
   - 后续安排

#### 沟通记录模板
```

一、基本信息
```

## 质量标准

### 沟通质量
1. **清晰性** — 是否清晰表达
2. **完整性** — 是否完整告知
3. **及时性** — 是否及时沟通
4. **有效性** — 是否有效沟通

### 风险告知质量
1. **全面性** — 是否全面告知
2. **客观性** — 是否客观告知
3. **及时性** — 是否及时告知
4. **可理解性** — 是否可理解
