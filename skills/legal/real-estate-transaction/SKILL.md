# 房产交易知识技能

## 一、交易流程

### 1.1 二手房交易流程

```
1. 看房、议价
2. 签订买卖合同
3. 网签备案
4. 资金监管
5. 缴纳税费
6. 办理过户
7. 领取新产权证
8. 交房、物业交割
```

### 1.2 新房交易流程

```
1. 看房、选房
2. 签订认购书
3. 签订正式合同
4. 网签备案
5. 办理贷款
6. 缴纳税费
7. 办理产权登记
8. 交房验收
```

## 二、税费计算

### 2.1 住宅交易税费

**买方**：
```
契税：
- 首套90㎡以下：1%
- 首套90㎡以上：1.5%
- 二套：1%-2%（各地不同）
- 三套及以上：3%

中介费：1%-2%
```

**卖方**：
```
增值税：
- 不满2年：全额 ÷ 1.05 × 5%
- 满2年：免征

个人所得税：
- 满五唯一：免征
- 不满五或不唯一：差额 × 20% 或 全额 × 1%

中介费：1%-2%
```

### 2.2 商铺交易税费

**买方**：
```
契税：3%
印花税：0.05%
```

**卖方**：
```
增值税：差额 ÷ 1.05 × 5%
土地增值税：30%-60%（四级超率累进）
个人所得税：差额 × 20%
印花税：0.05%
```

### 2.3 速查表

```
满五唯一住宅：
- 买方：契税1%-1.5%
- 卖方：增值税免 + 个税免
- 总税费：约1%-1.5%

不满2年住宅：
- 买方：契税1%-3%
- 卖方：增值税5% + 个税1%
- 总税费：约7%-9%

商铺：
- 买方：契税3% + 印花税0.05%
- 卖方：增值税5% + 土地增值税30%-60% + 个税20% + 印花税0.05%
- 总税费：较高
```

## 三、限购政策

### 3.1 主要城市限购（2024-2025）

| 城市 | 限购范围 | 条件 |
|------|---------|------|
| 北京 | 五环内 | 本地户籍2套，非本地1套（5年社保） |
| 上海 | 外环内 | 本地户籍2套，非本地1套（5年社保） |
| 广州 | 全市 | 本地户籍2套，非本地1套（5年社保） |
| 深圳 | 全市 | 本地户籍2套，非本地1套（5年社保） |

**注**：各城市政策持续调整，需查询最新规定。

### 3.2 限贷政策

```
首套：
- 首付比例：20%-30%
- 利率：LPR-20BP至LPR

二套：
- 首付比例：30%-70%
- 利率：LPR+20BP至LPR+60BP
```

## 四、常见纠纷

### 4.1 合同效力

**有效情形**：
- 双方意思表示真实
- 不违反法律强制性规定
- 不违背公序良俗

**无效情形**：
- 无权处分（但不影响善意取得）
- 违反限购政策（合同效力有争议）
- 恶意串通损害他人利益

### 4.2 违约责任

**卖方违约**：
- 一房二卖：返还房款 + 赔偿损失 + 惩罚性赔偿
- 拒绝过户：继续履行 + 违约金

**买方违约**：
- 拒绝付款：没收定金 + 违约金
- 逾期付款：逾期利息 + 违约金

### 4.3 典型案例

**一房二卖**：
- 先买方：可要求继续履行或解除合同 + 赔偿损失
- 后买方：如已过户且善意取得，可获得产权
- 卖方：承担惩罚性赔偿责任

## 五、产权登记

### 5.1 登记类型

- 首次登记：新建房屋
- 转移登记：买卖、赠与、继承
- 变更登记：面积、用途变更
- 注销登记：房屋灭失

### 5.2 登记效力

- 不动产物权的设立、变更、转让和消灭，经依法登记，发生效力
- 未经登记，不发生效力（但法律另有规定的除外）

**法律依据**：《民法典》第209条
