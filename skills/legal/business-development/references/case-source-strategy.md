# 案源突破策略

## 一、案源少的根本原因

| 问题 | 表现 | 解决方向 |
|------|------|---------|
| 曝光不够 | 没人知道你是律师 | 增加曝光渠道 |
| 信任不足 | 知道你但不敢委托 | 建立专业形象 |
| 定位不清 | 什么案都接，没有特色 | 打造专业标签 |
| 渠道单一 | 只靠熟人介绍 | 多渠道获客 |

## 二、线上获客

### 短视频/直播（推荐指数⭐⭐⭐⭐⭐）

**平台**：抖音、视频号、小红书

**内容方向**：
- 法律科普："借钱不还怎么办"
- 热点解读："XX明星离婚案的法律看点"
- 案例故事："我帮当事人要回了50万"
- 反常识知识："签了合同也能反悔的5种情况"

**关键指标**：
- 坚持日更或隔日更
- 前3个月不要期望回报
- 100个视频是起步线

**变现路径**：
短视频 → 直播咨询 → 私信转化 → 线下面谈 → 委托

### 自媒体文章（推荐指数⭐⭐⭐⭐）

**平台**：微信公众号、知乎、头条

**内容方向**：
- 深度法律分析文章
- 案例解析（脱敏后）
- 法律热点评论

**关键指标**：
- 每周2-3篇
- SEO优化标题
- 引导关注和私信

### 法律问答平台（推荐指数⭐⭐⭐）

**平台**：
- 华律网
- 找法网
- 法律快车
- 百度知道法律板块

**策略**：
- 每天回答5-10个问题
- 选择本地+专业领域的问题
- 回答要专业、详细、有温度

## 三、线下获客

### 社区/企业普法讲座（推荐指数⭐⭐⭐⭐⭐）

**形式**：
- 社区法律咨询日
- 企业法律培训
- 行业协会讲座

**关键点**：
- 免费或低价
- 准备名片和宣传页
- 现场收集联系方式
- 后续跟进

### 商会/协会/社团（推荐指数⭐⭐⭐⭐）

**加入**：
- 本地商会
- 行业协会
- 校友会
- 同乡会

**策略**：
- 主动提供法律服务
- 成为法律顾问
- 参与活动建立人脉

### 异业合作（推荐指数⭐⭐⭐⭐）

**合作对象**：
- 会计师/税务师 → 财税纠纷案源
- 房产中介 → 房产纠纷案源
- 保险代理人 → 保险纠纷案源
- 公证员 → 继承、赠与案源
- 婚姻介绍所 → 离婚案源
- 工商代办 → 公司纠纷案源

**合作模式**：
- 互相推荐
- 联合服务
- 分成机制

## 四、老客户转介绍

### 核心策略

```
1. 案件办得好 → 客户满意
2. 服务态度好 → 客户感动
3. 主动要求转介绍 → 客户推荐
4. 维护老客户关系 → 持续转介绍
```

### 具体做法

- 案件结束后主动问："身边有朋友需要法律服务可以找我"
- 定期回访老客户（节日问候、生日祝福）
- 建立客户微信群，定期分享法律知识
- 转介绍成功后表示感谢（礼品、请客）

## 五、专业化定位

### 为什么要做专业化？

| 万金油律师 | 专业律师 |
|----------|---------|
| 什么案都接 | 专注1-2个领域 |
| 客户不知道你擅长什么 | 客户知道你专业 |
| 价格竞争 | 专业溢价 |
| 获客难 | 口碑获客 |

### 如何选择专业方向？

**考虑因素**：
- 本地市场需求
- 个人兴趣和专长
- 竞争对手数量
- 收费空间

**推荐方向**（江苏市场）：
- 劳动纠纷（需求大、门槛适中）
- 建设工程（收费高、专业性强）
- 知识产权（增长快、蓝海市场）
- 公司股权（企业刚需）

## 六、AI赋能获客

利用AI技能提高效率：

| 能力 | 获客应用 |
|------|---------|
| 法律检索秒级响应 | 直播中实时回答观众问题 |
| 企业信息查询 | 帮客户做尽调，展示专业能力 |
| 财务/税务分析 | 提供增值服务，区别于其他律师 |
| 文书自动生成 | 提高效率，服务更多客户 |
| 违禁词检测 | 直播合规，避免封号 |

**建议**：
- 用AI技能提高效率 → 服务更多客户
- 用AI能力做直播内容 → 吸引更多关注
- 用AI工具做增值服务 → 提高客单价
