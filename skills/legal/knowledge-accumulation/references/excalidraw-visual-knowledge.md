# Excalidraw 可视化知识图谱模式

当知识体系涉及复杂关系时，使用 Excalidraw 创建可视化图表辅助学习和记忆。

## 何时使用

- 概念之间有复杂关系（如五行生克、十神体系）
- 学习路径需要可视化展示（如分阶段流程）
- 业务场景需要图解（如客户类型对照）
- 速记内容需要图形化（如速查表）

## 图表类型选择

| 图表类型 | 适用场景 | 布局方式 |
|----------|----------|----------|
| 流程图 | 学习路径、工作流程 | 左→右或上→下 |
| 关系图 | 概念关系、生克关系 | 中心辐射 |
| 场景图 | 业务应用、对话示例 | 分区卡片 |
| 速记图 | 知识点速查 | 网格/表格 |

## 创建流程

1. **设计布局** — 先想清楚图表结构
2. **创建 .excalidraw 文件** — 使用 `excalidraw` skill
3. **上传获取链接** — `python scripts/upload.py <file>`
4. **嵌入 Obsidian** — 文档中附链接或直接嵌入文件

## Excalidraw 要点

- 使用 `containerId` + `boundElements` 绑定文字到形状
- 每个元素的 `boundText` 紧跟其容器元素后面（z-order）
- 颜色使用 `references/colors.md` 中的调色板
- 最小 `fontSize: 16`，标题用 `fontSize: 20-32`
- 不要在文字中使用 emoji（Excalidraw 字体不支持）
