## 二、财务报表分析

### 2.1 资产负债表分析

**关键指标**：
```
资产负债率 = 总负债 / 总资产 × 100%
  - < 40%：财务稳健
  - 40%-60%：适中
  - > 60%：财务风险较高
  - > 70%：高风险

流动比率 = 流动资产 / 流动负债
  - > 2：短期偿债能力强
  - 1-2：适中
  - < 1：短期偿债压力大

速动比率 = (流动资产 - 存货) / 流动负债
  - > 1：短期偿债能力强
  - < 1：短期偿债压力大
```

**法律应用场景**：
- 债务纠纷：评估债务人偿债能力
- 股权收购：评估目标公司财务状况
- 破产申请：判断是否资不抵债

### 2.2 利润表分析

**关键指标**：
```
毛利率 = (营业收入 - 营业成本) / 营业收入 × 100%
  - 反映产品盈利能力

净利率 = 净利润 / 营业收入 × 100%
  - 反映综合盈利能力

营业收入增长率 = (本期 - 上期) / 上期 × 100%
  - 反映成长性
```

**法律应用场景**：
- 财务造假：毛利率异常偏高可能是虚增收入
- 合同纠纷：计算可得利益损失
- 股权估值：预测未来收益

### 2.3 现金流量表分析

**三类现金流**：
```
经营活动现金流：核心业务产生
投资活动现金流：买卖资产产生
筹资活动现金流：借款还款产生
```

**关键判断**：
- 经营现金流为正：主业健康
- 经营现金流为负：主业亏损（警惕）
- 经营现金流远小于净利润：可能虚增利润

**法律应用场景**：
- 财务造假识别：净利润与现金流严重背离
- 偿债能力评估：现金流是否足以还债
- 持续经营判断：现金流是否能维持运营

## 三、财务造假识别

### 3.1 常见造假手法

| 手法 | 表现 | 识别方法 |
|------|------|---------|
| 虚增收入 | 收入大幅增长，现金流不匹配 | 收入/现金流比率异常 |
| 虚减成本 | 毛利率异常偏高 | 与同行对比 |
| 虚增资产 | 存货/应收账款异常增长 | 周转率下降 |
| 隐藏负债 | 表外负债、或有负债 | 关注附注披露 |
| 关联交易 | 非公允价格交易 | 关注关联方披露 |
| 提前确认收入 | 收入确认时点异常 | 关注收入确认政策 |

### 3.2 财务造假法律后果

**民事责任**：
- 《证券法》第85条：信息披露义务人未按照规定披露信息，致使投资者在证券交易中遭受损失的，应当承担赔偿责任
- 《公司法》第20条：公司股东滥用公司法人独立地位和股东有限责任，逃避债务，严重损害公司债权人利益的，应当对公司债务承担连带责任

**行政责任**：
- 《证券法》第197条：信息披露义务人未按照规定披露信息，责令改正，给予警告，并处以罚款

**刑事责任**：
- 《刑法》第161条：违规披露、不披露重要信息罪
- 《刑法》第160条：欺诈发行股票、债券罪

### 3.3 典型案例

**康美药业财务造假案**：
- 虚增货币资金887亿元
- 虚增收入291亿元
- 虚增利润39亿元
- 判决：赔偿投资者24.59亿元

**瑞幸咖啡财务造假案**：
- 虚增收入22亿元
- 虚增成本13亿元
- 处罚：罚款1.8亿美元

## 四、审计实务

### 4.1 审计责任

**审计机构责任**：
- 《注册会计师法》第21条：注册会计师执行审计业务，必须按照执业准则、规则确定的工作程序出具报告
- 《证券法》第163条：证券服务机构为证券的发行、上市、交易等证券业务活动制作、出具审计报告等文件，应当勤勉尽责

**审计失败法律后果**：
- 民事赔偿：与被审计单位承担连带赔偿责任
- 行政处罚：警告、罚款、暂停执业、吊销执照
- 刑事处罚：提供虚假证明文件罪

### 4.2 审计报告类型

| 类型 | 含义 | 法律风险 |
|------|------|---------|
| 无保留意见 | 财务报表公允反映 | 低 |
| 保留意见 | 部分事项存疑 | 中 |
| 否定意见 | 财务报表存在重大错报 | 高 |
| 无法表示意见 | 审计范围受限 | 高 |

**法律应用场景**：
- 审计责任纠纷：审计报告类型是判断审计机构是否尽责的重要依据
- 财务造假诉讼：审计报告类型影响投资者损失认定

### 4.3 审计程序

```
1. 风险评估：了解被审计单位及其环境
2. 控制测试：测试内部控制有效性
3. 实质性程序：对各类交易、账户余额实施审计程序
4. 完成审计工作：汇总审计差异、获取管理层声明
5. 出具审计报告：形成审计意见
```

## 五、企业估值

### 5.1 估值方法

| 方法 | 公式 | 适用场景 |
|------|------|---------|
| 收益法 | 企业价值 = Σ(未来现金流 / (1+折现率)^n) | 持续经营企业 |
| 市场法 | 企业价值 = 可比公司市盈率 × 目标公司净利润 | 上市公司 |
| 资产基础法 | 企业价值 = 资产评估值 - 负债评估值 | 清算企业 |

### 5.2 股权估值争议

**常见争议**：
- 估值方法选择：收益法 vs 市场法 vs 资产基础法
- 折现率确定：无风险利率 + 风险溢价
- 收益预测：未来收益的合理性
- 控制权溢价：控股股东 vs 少数股东

**法律依据**：
- 《公司法》第71条：股权转让
- 《公司法》第74条：异议股东回购请求权
- 《公司法》第142条：股份回购

## 六、内部控制

### 6.1 内部控制五要素

```
1. 控制环境：管理层诚信、治理结构
2. 风险评估：识别经营风险
3. 控制活动：审批、授权、核对
4. 信息与沟通：财务报告系统
5. 监督：内部审计
```

### 6.2 内部控制缺陷

| 缺陷类型 | 定义 | 法律后果 |
|----------|------|---------|
| 重大缺陷 | 可能导致重大错报未被防止或发现 | 必须披露 |
| 重要缺陷 | 严重程度低于重大缺陷但值得关注 | 应当披露 |
| 一般缺陷 | 严重程度低于重要缺陷 | 内部处理 |

**法律应用场景**：
- 公司治理纠纷：内部控制缺陷是判断管理层是否尽责的依据
- 财务造假诉讼：内部控制缺陷可能导致财务造假未被发现

## 七、速查表

### 7.1 财务造假识别速查

```
异常信号：
1. 收入大幅增长但现金流不匹配
2. 毛利率异常偏高（远超同行）
3. 应收账款异常增长
4. 存货异常增长
5. 经营现金流与净利润严重背离
6. 频繁变更会计政策
7. 关联交易异常
8. 大额资产减值后又转回
```

### 7.2 偿债能力评估速查

```
短期偿债：
- 流动比率 > 2：安全
- 速动比率 > 1：安全
- 现金比率 > 0.2：安全

长期偿债：
- 资产负债率 < 60%：安全
- 利息保障倍数 > 3：安全
- 经营现金流/总负债 > 0.2：安全
```

### 7.3 股权估值速查

```
上市公司：
- 市盈率法：企业价值 = 净利润 × 市盈率
- 市净率法：企业价值 = 净资产 × 市净率

非上市公司：
- 收益法：企业价值 = Σ(未来现金流 / (1+折现率)^n)
- 资产基础法：企业价值 = 资产评估值 - 负债评估值
```
