# 注册税务师（CTA）知识技能

## 核心定位

为律师处理涉税案件提供专业知识支持——税法体系、各税种计算、税务筹划、税务稽查、涉税争议解决。

## 一、税法体系

### 1.1 中国税种分类

| 类别 | 税种 | 法律依据 |
|------|------|---------|
| **流转税** | 增值税、消费税、关税 | 增值税法、消费税法 |
| **所得税** | 企业所得税、个人所得税 | 企业所得税法、个人所得税法 |
| **财产税** | 房产税、契税、车船税 | 房产税暂行条例、契税法 |
| **行为税** | 印花税、城市维护建设税 | 印花税法 |
| **资源税** | 资源税、土地增值税 | 资源税法、土地增值税暂行条例 |
| **特定目的税** | 环境保护税、烟叶税 | 环境保护税法 |

### 1.2 税法效力层级

```
宪法 → 税收法律 → 税收行政法规 → 税收部门规章 → 税收规范性文件
```

**重要税收法律**：
- 《税收征收管理法》（程序法）
- 《个人所得税法》
- 《企业所得税法》
- 《增值税法》（2026年1月1日起施行）

## 二、增值税

### 2.1 税率

| 类别 | 税率 | 适用范围 |
|------|------|---------|
| 基本税率 | 13% | 销售货物、加工修理修配 |
| 低税率 | 9% | 交通运输、建筑、农产品 |
| 低税率 | 6% | 现代服务、金融服务 |
| 零税率 | 0% | 出口货物 |

### 2.2 计算公式

```
一般纳税人：
应纳税额 = 当期销项税额 - 当期进项税额
销项税额 = 销售额 × 税率

小规模纳税人：
应纳税额 = 销售额 × 征收率（3%）
```

### 2.3 常见法律场景

**虚开增值税专用发票罪**：
- 《刑法》第205条
- 虚开税款数额1万元以上或者致使国家税款被骗取5000元以上的，应当依法定罪处罚
- 虚开税款数额50万元以上的，属于"数额较大"
- 虚开税款数额250万元以上的，属于"数额巨大"

**法律后果**：
- 3年以下有期徒刑或者拘役，并处2万元以上20万元以下罚金
- 数额较大：3年以上10年以下有期徒刑，并处5万元以上50万元以下罚金
- 数额巨大：10年以上有期徒刑或者无期徒刑，并处5万元以上50万元以下罚金或者没收财产

## 三、企业所得税

### 3.1 税率

| 类型 | 税率 | 适用范围 |
|------|------|---------|
| 基本税率 | 25% | 一般企业 |
| 小型微利企业 | 20% | 年应纳税所得额≤300万元 |
| 高新技术企业 | 15% | 国家重点扶持高新技术企业 |
| 西部大开发 | 15% | 西部地区鼓励类产业 |

### 3.2 计算公式

```
应纳税所得额 = 收入总额 - 不征税收入 - 免税收入 - 扣除项目 - 允许弥补的以前年度亏损

应纳税额 = 应纳税所得额 × 税率 - 减免税额 - 抵免税额
```

### 3.3 常见扣除项目

| 项目 | 扣除标准 | 法律依据 |
|------|---------|---------|
| 职工福利费 | 工资总额14% | 企业所得税法实施条例第40条 |
| 工会经费 | 工资总额2% | 企业所得税法实施条例第41条 |
| 职工教育经费 | 工资总额8% | 企业所得税法实施条例第42条 |
| 业务招待费 | 发生额60%且≤收入5‰ | 企业所得税法实施条例第43条 |
| 广告费和业务宣传费 | 收入15%（特殊行业30%） | 企业所得税法实施条例第44条 |
| 公益性捐赠 | 年度利润12% | 企业所得税法第9条 |

### 3.4 税收优惠

**小型微利企业**：
- 年应纳税所得额≤300万元
- 从业人数≤300人
- 资产总额≤5000万元
- 实际税负：5%（应纳税所得额≤100万部分）+ 10%（100万-300万部分）

**高新技术企业**：
- 税率：15%
- 研发费用加计扣除：100%
- 技术转让所得：500万元以下免税，500万元以上减半

## 四、个人所得税

### 4.1 税率表（综合所得）

| 级数 | 应纳税所得额 | 税率 | 速算扣除数 |
|------|------------|------|----------|
| 1 | ≤36,000 | 3% | 0 |
| 2 | 36,000-144,000 | 10% | 2,520 |
| 3 | 144,000-300,000 | 20% | 16,920 |
| 4 | 300,000-420,000 | 25% | 31,920 |
| 5 | 420,000-660,000 | 30% | 52,920 |
| 6 | 660,000-960,000 | 35% | 85,920 |
| 7 | >960,000 | 45% | 181,920 |

### 4.2 计算公式

```
综合所得应纳税额 = (年收入 - 60000 - 专项扣除 - 专项附加扣除 - 其他扣除) × 税率 - 速算扣除数

经营所得应纳税额 = (年收入 - 成本 - 费用 - 损失) × 税率 - 速算扣除数
```

### 4.3 专项附加扣除

| 项目 | 标准 | 条件 |
|------|------|------|
| 子女教育 | 2000元/月/子女 | 满3岁至博士毕业 |
| 继续教育 | 400元/月或3600元/年 | 学历教育或职业资格 |
| 大病医疗 | 实际支出15000-80000元 | 医保目录内自付部分 |
| 住房贷款利息 | 1000元/月 | 首套房贷，最长240个月 |
| 住房租金 | 800-1500元/月 | 工作城市无房 |
| 赡养老人 | 3000元/月 | 60岁以上父母 |
| 婴幼儿照护 | 2000元/月/婴幼儿 | 3岁以下 |

### 4.4 股权转让个人所得税

```
应纳税额 = (转让收入 - 股权原值 - 合理费用) × 20%

转让收入：按照公平交易原则确定
股权原值：实际出资额或取得股权时的公允价值
合理费用：印花税、中介费等
```

**法律依据**：
- 《个人所得税法》第6条
- 《股权转让所得个人所得税管理办法（试行）》（国家税务总局公告2014年第67号）

**常见争议**：
- 转让价格明显偏低且无正当理由
- 股权原值确定
- 纳税义务发生时间

## 五、其他重要税种

### 5.1 契税

```
税率：3%-5%（各省自定）
计税依据：不动产成交价格

应纳税额 = 不动产成交价格 × 税率

优惠：
- 首套房90㎡以下：1%
- 首套房90㎡以上：1.5%
- 二套房：1%-2%
```

### 5.2 土地增值税

```
税率：30%-60%（四级超率累进税率）

增值额 = 转让收入 - 扣除项目金额
增值率 = 增值额 / 扣除项目金额 × 100%

| 增值率 | 税率 | 速算扣除系数 |
|--------|------|------------|
| ≤50% | 30% | 0 |
| 50%-100% | 40% | 5% |
| 100%-200% | 50% | 15% |
| >200% | 60% | 35% |
```

### 5.3 印花税

```
常见税目税率：
- 买卖合同：价款的0.03%
- 借款合同：借款金额的0.005%
- 财产租赁合同：租金的0.1%
- 股权转让书据：转让金额的0.05%
- 营业账簿：实收资本+资本公积的0.025%
```

## 六、税务筹划

### 6.1 企业税务筹划

**组织形式选择**：
- 有限公司：企业所得税25% + 个人所得税20%（分红）
- 个人独资企业：个人所得税5%-35%
- 合伙企业：个人所得税5%-35%（先分后税）

**区域优惠利用**：
- 西部大开发：15%企业所得税
- 海南自贸港：15%企业所得税
- 横琴/前海/平潭：15%企业所得税

**行业优惠利用**：
- 高新技术企业：15%企业所得税
- 研发费用加计扣除：100%
- 技术转让所得：500万以下免税

### 6.2 个人税务筹划

**工资薪金筹划**：
- 充分利用专项附加扣除
- 合理安排年终奖（单独计税 vs 并入综合所得）
- 利用商业健康保险、税收递延型养老保险

**股权转让筹划**：
- 先分红后转让（减少转让收入）
- 利用正当理由低价转让
- 分次转让（利用小额免税政策）

## 七、税务稽查

### 7.1 税务稽查流程

```
1. 选案：随机抽查、举报、数据分析异常
2. 检查：调取账簿、询问、实地检查
3. 审理：事实认定、法律适用、处罚决定
4. 执行：追缴税款、滞纳金、罚款
```

### 7.2 税务行政处罚

| 违法行为 | 处罚 | 法律依据 |
|----------|------|---------|
| 偷税 | 50%-5倍罚款 | 税收征管法第63条 |
| 逃避追缴欠税 | 50%-5倍罚款 | 税收征管法第65条 |
| 骗取出口退税 | 1-5倍罚款 | 税收征管法第66条 |
| 抗税 | 1-5倍罚款 | 税收征管法第67条 |
| 未按规定申报 | 2000元以下罚款 | 税收征管法第62条 |

### 7.3 涉税刑事犯罪

| 罪名 | 构成要件 | 刑罚 |
|------|---------|------|
| 逃税罪 | 逃避缴纳税款，数额较大 | 3年以下或3-7年 |
| 逃避追缴欠税罪 | 转移财产逃避欠税 | 3年以下或3-7年 |
| 骗取出口退税罪 | 骗取出口退税 | 5年以下至无期 |
| 虚开增值税专用发票罪 | 虚开发票 | 3年以下至无期 |
| 非法出售增值税专用发票罪 | 出售发票 | 3年以下至无期 |

## 八、涉税争议解决

### 8.1 税务行政复议

**前置条件**：
- 必须先缴纳税款或提供担保
- 对征税行为不服的，必须先复议后诉讼

**复议期限**：
- 知道具体行政行为之日起60日内
- 对征税行为不服的，先缴清税款和滞纳金或提供担保后60日内

### 8.2 税务行政诉讼

**诉讼期限**：
- 对复议决定不服的，收到复议决定书之日起15日内
- 复议机关逾期不作决定的，自期满之日起15日内

**举证责任**：
- 被告（税务机关）对作出的行政行为负有举证责任

### 8.3 偷税辩护要点

**主观故意**：
- 偷税必须有主观故意
- 因计算错误等失误未缴或少缴税款不构成偷税

**客观行为**：
- 伪造、变造、隐匿、擅自销毁账簿、记账凭证
- 在账簿上多列支出或者不列、少列收入
- 经税务机关通知申报而拒不申报或者进行虚假的纳税申报

**辩护策略**：
- 主观上无偷税故意
- 客观上未实施偷税行为
- 计算错误而非故意
- 已补缴税款、滞纳金、罚款

## 九、速查表

### 9.1 常见税种税率速查

```
增值税：13% / 9% / 6% / 0%
企业所得税：25%（一般）/ 15%（高新技术）/ 20%（小微）
个人所得税：3%-45%（综合所得）/ 5%-35%（经营所得）
契税：3%-5%
印花税：0.005%-0.1%
土地增值税：30%-60%
```

### 9.2 股权转让税费速查

```
个人所得税：(转让收入 - 原值 - 费用) × 20%
企业所得税：(转让收入 - 原值) × 25%
印花税：转让金额 × 0.05%
```

### 9.3 房产交易税费速查

```
买方：
- 契税：1%-3%
- 印花税：0.05%

卖方：
- 增值税：5%（不足2年）/ 0%（满2年）
- 个人所得税：差额20%或全额1%
- 土地增值税：30%-60%（个人住房免征）
- 印花税：0.05%
```
