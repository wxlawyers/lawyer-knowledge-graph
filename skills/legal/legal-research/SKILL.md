# 法律检索与案例研究技能

面向中国诉讼律师的法律检索框架，集成元典智库和北大法宝MCP工具，提供三轮递进检索策略。

## 适用场景

- 请求权基础检索
- 类案检索
- 裁判规则验证
- 法规时效性确认
- 法条内容查询

## 执行步骤

### 步骤一：确定检索目标

#### 检索类型
1. **请求权基础检索**
   - 确定法律关系性质
   - 检索相关法规
   - 检索具体法条

2. **类案检索**
   - 确定案由和争议焦点
   - 检索类似案例
   - 提取裁判规则

3. **裁判规则验证**
   - 验证规则适用性
   - 对比我方案情
   - 判断参考价值

#### 检索关键词
1. **案由关键词**
   - 合同纠纷
   - 侵权纠纷
   - 物权纠纷
   - 公司纠纷

2. **争议焦点关键词**
   - 违约责任
   - 侵权责任
   - 物权变动
   - 股东权利

3. **地域关键词**
   - 省份名称
   - 城市名称
   - 法院名称

4. **时间关键词**
   - 年份
   - 时间段

### 步骤二：第一轮检索（双源交叉）

#### 元典智库检索

**法规检索**
```
mcp_yuandian_search_fagui(keyword="民法典")
```

**法条检索**
```
mcp_yuandian_search_fatiao(keyword="民法典 第五百七十七条")
```

**案例检索**
```
mcp_yuandian_search_qwal(keyword="合同纠纷 违约 南京 2024")
```

#### 北大法宝检索

**法规搜索**
```
mcp_pkulaw_search_law(keyword="民法典")
```

**案例检索**
```
mcp_pkulaw_search_case(keyword="合同纠纷")
```

**法条检索**
```
mcp_pkulaw_search_fatiao(keyword="民法典 第五百七十七条")
```

#### 交叉验证
1. **对比两个平台的检索结果**
   - 法规是否一致
   - 法条内容是否一致
   - 案例是否一致

2. **标注差异**
   - 记录不同之处
   - 分析原因
   - 确认权威来源

### 步骤三：第二轮检索（裁判规则验证）

#### 提取裁判规则
1. **从案例中提取**
   - 裁判要旨
   - 裁判理由
   - 裁判结果

2. **归纳规则**
   - 提炼核心规则
   - 确定适用条件
   - 明确法律依据

#### 验证规则适用性
1. **对比案情**
   - 事实是否相似
   - 法律关系是否相同
   - 争议焦点是否一致

2. **判断参考价值**
   - 案例级别（最高法/高级法院/中级法院）
   - 案例时间（是否最新）
   - 案例地域（是否同一地区）

### 步骤四：第三轮检索（类案补充）

#### 检索策略
1. **同一法院历史裁判**
   - 检索同一法院的类似案例
   - 分析裁判倾向
   - 预判裁判结果

2. **同一法官历史裁判**
   - 检索同一法官的类似案例
   - 分析裁判风格
   - 预判裁判倾向

3. **同一地区中级法院**
   - 检索同一地区中级法院案例
   - 分析裁判标准
   - 预判上诉结果

#### 检索关键词组合
1. **案由 + 争议焦点**
   - 合同纠纷 + 违约责任
   - 侵权纠纷 + 因果关系
   - 物权纠纷 + 物权变动
   - 公司纠纷 + 股东权利

2. **案由 + 地域 + 年份**
   - 合同纠纷 + 南京 + 2024
   - 侵权纠纷 + 江苏 + 2023
   - 物权纠纷 + 无锡 + 2024

3. **争议焦点 + 法律依据**
   - 违约责任 + 民法典第五百七十七条
   - 侵权责任 + 民法典第一千一百六十五条
   - 物权保护 + 民法典第二百三十五条

### 步骤五：检索结果处理

#### 法规检索结果
1. **确认时效性**
   - 是否现行有效
   - 是否被修改
   - 是否被废止

2. **确认完整性**
   - 条文是否完整
   - 是否有司法解释补充
   - 是否有指导性案例

3. **标注来源**
   - 检索平台
   - 检索时间
   - 检索关键词

#### 案例检索结果
1. **确认有效性**
   - 是否被撤销
   - 是否被改判
   - 是否被再审

2. **提取裁判要旨**
   - 核心裁判观点
   - 裁判理由
   - 法律依据

3. **判断参考价值**
   - 案例级别
   - 案例时间
   - 案例地域

#### 法条检索结果
1. **确认内容完整性**
   - 条文是否完整
   - 是否有司法解释补充
   - 是否有指导性案例

2. **标注来源**
   - 检索平台
   - 检索时间
   - 检索关键词

## 检索策略

### 策略一：请求权基础检索

**目的**：确定请求权基础的法律依据

**步骤**：
1. 确定法律关系性质（合同/侵权/物权/公司）
2. 检索相关法规（民法典合同编/侵权责任编/物权编/公司法）
3. 检索具体法条（违约责任/侵权责任/物权保护/股东权利）
4. 交叉验证法条有效性

### 策略二：类案检索

**目的**：查找支持我方主张的类案

**步骤**：
1. 确定案由和争议焦点
2. 组合关键词检索（案由+争议焦点+地域+年份）
3. 提取裁判要旨
4. 验证案例有效性

### 策略三：裁判规则验证

**目的**：验证裁判规则的适用性

**步骤**：
1. 检索相关案例
2. 提取裁判要旨
3. 对比我方案情
4. 判断是否适用

## 质量标准

### 检索质量
1. **全面性** — 是否覆盖所有相关法规和案例
2. **准确性** — 检索结果是否准确
3. **时效性** — 法规是否现行有效
4. **权威性** — 案例是否具有参考价值

### 引用规范
1. **法条引用** — 全称+条号+内容
2. **案例引用** — 案号+法院+裁判要旨
3. **来源标注** — 检索平台+检索时间

## 安全要求

- 当事人敏感信息脱敏处理
- 检索结果仅供律师参考
- 不作为法律意见对外出具
